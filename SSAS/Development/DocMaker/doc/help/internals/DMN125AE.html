Portable Document Format

The Adobe portable document format (PDF) is a popular format for distributing high-quality documentation. Files can be opened using the free Adobe Acrobat Reader application, or other compatible readers. PDF documents differ from HTML documents in their presentation; unlike HTML, PDF documents are paginated and are ready for printing to paper if required. DocMaker supports many PDF features to make these documents as useful as possible: the table of contents is generated automatically; bookmarks are created to allow the PDF reader application to display the table of contents independently beside the document content; all entries in the table of contents, all cross-references and all external references are active hyperlinks; headers and footers with titles and page numbers are inserted; and cover pages and color schemes are selected automatically to match the category of the document. All of these features are supported automatically by DocMaker and only the latter feature requires any configuration. To modify the cover page and color scheme, you must set the role attribute of the book element; see the section, Book Categories for more details.
