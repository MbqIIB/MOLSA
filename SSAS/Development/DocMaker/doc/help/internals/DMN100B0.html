Installing DocMaker

Introduction

DocMaker is released internally in Cúram in a different form to that released to customers. Customers use DocMaker via the Documentation Center development environment; they do not install DocMaker as a separate product. Within Cúram, DocMaker releases are made available to support its integration with the Documentation Center or for use in automated build scripts. Other internal users may just be interested in installing and running DocMaker as a separate product for testing purposes. This chapter describes the installation steps for the internal release of DocMaker and includes details on what it contains and how it can be integrated into build scripts and other systems. This information is relevant to internal users only.

Some of the tools developed for DocMaker can also be used in other projects. See DocMaker Tools for more details.
