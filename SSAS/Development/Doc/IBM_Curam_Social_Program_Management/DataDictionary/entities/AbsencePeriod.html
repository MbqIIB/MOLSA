Database Table: AbsencePeriod

Table Description
Absence Period: Detailed information about absences and units unattended for a period for a client.

This table stores detailed information about absences and units unattended for a period for a client.

Example

For example, a child was scheduled to receive 80 units of Child Care between 01/05/2008-08/05/2008 but only received 60 units and 20 units remained unattended due to various absence reasons.
