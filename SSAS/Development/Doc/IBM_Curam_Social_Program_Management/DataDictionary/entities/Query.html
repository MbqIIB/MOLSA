Database Table: Query

Table Description
Query: A saved user defined database query.

A user has the ability to create queries based on criteria which suits their needs. Queries are saved and can then be easily accessed by the user. This entity captures the details of the saved query.

Example

For example, a caseworker routinely searches for all Appeals cases assigned to themselves that are in an Open status and were created in the last month. The query will be saved against the users profile and available each time they login. The query can be re-run with a single click saving the user from having to re-enter the search criteria each time.
