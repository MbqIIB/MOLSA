Database Table: OrganisationStructure

Table Description
Organization Structure: The top level of the organization hierarchy.

An organization structure contains organization units that have positions. A user can hold one or more positions. An organization structure represents the configuration of the organization for both workflow and reporting purposes.

Example

Midway Social Services is a Human Services organization which has implemented Curam. An organization structure is created that contains the organization hierarchy. Within the structure five organization units are created each of which contains a number of positions.
