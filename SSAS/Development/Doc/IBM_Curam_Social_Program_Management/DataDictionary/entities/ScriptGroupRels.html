Database Table: ScriptGroupRels

Table Description
Script Group Relationship: A Question Group used by an IEG Script.

This table is used by the IEG Editor to identify which IEG Scripts use a particular Question Group. This enables the IEG Editor to:

  • warn a User who is modifying a Question Group that the group is used by an IEG Script; and
  • prevent a User from deleting a Question Group which is used by any IEG Script.

Example

You implement new application functionality to screen claims for your unemployment Product. Your functionality requires to execute an IEG Script to gather basic evidence for the claim.

You use the IEG Editor to define a new IEG Script to gather this evidence. Your IEG Script contains the following Question Groups:

  • personal details - the claimant's name, address and date of birth;
  • cessation reason - the reason why the claimant left their last employment; and
  • last employment details - the name and address of the claimant's last employer and the date that the claimant's employment ceased.

You later change the personal details Question Group to include 'number of dependents'. The IEG Editor reads this table and warns you that the Question Group is used by your unemployment screening script.
