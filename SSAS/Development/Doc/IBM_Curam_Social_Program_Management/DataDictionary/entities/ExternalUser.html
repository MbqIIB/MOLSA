Database Table: ExternalUser

Table Description
External User: An external user is a user of the system who is not part of the organization structure. They access the application over the web and have restricted access to the system. External User accounts are managed separately to those of internal users.

Example

An employer is assigned an external user account so that they can connect to the agency's portal and see what data the agency has in relation to the employer, their employee's etc

Technical Notes

Curam does not ship with any external user accounts
