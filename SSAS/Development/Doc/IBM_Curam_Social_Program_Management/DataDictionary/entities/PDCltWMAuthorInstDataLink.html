Database Table: PDCltWMAuthorInstDataLink

Table Description
ProductDeliveryClientWMAuthorisationInstanceDataLink : Provides the client details which will be added to a product delivery created as a result of the linked authorisation instance. The clients persisted here are as a result of a system user selecting them. This entity captures the clients and their roles, as specified by the user, that will be added to the product delivery created as part of the deferred authorization process.
