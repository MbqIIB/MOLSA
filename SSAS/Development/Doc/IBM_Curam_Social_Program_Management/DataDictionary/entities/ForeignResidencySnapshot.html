Database Table: ForeignResidencySnapshot

Table Description
Foreign Residency Snapshot: Maintains a history of changes to a foreign residency record.

Example

If a user modifies, for example, the reason for foreign residency of a participant, a snapshot of the foreign residency data prior to making this modification is written to the foreign residency snapshot table. Likewise when any of the foreign residency details are modified a snapshot of the previous version of the record is created.
