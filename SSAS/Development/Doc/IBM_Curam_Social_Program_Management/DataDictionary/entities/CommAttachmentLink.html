Database Table: CommAttachmentLink

Table Description
Communication Attachment Link: A link between an attachment and a communication.

This table allows the sharing and reuse of multiple attachments across multiple communications.

Example

An informational leaflet is sent to John Smith and Linda Smith advising them of changes to the payment rates for a program called 'sickness benefit' rated for the following year. A copy of this leaflet is stored as an attachment on the communication for both participants.
