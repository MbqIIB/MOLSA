Database Table: ProductAppealProcess

Table Description
Product Appeal Process: The appeal process definition for a product.

A product appeal process is composed of one or more stages. In addition, a single product can have one or more appeal processes, each with a different start date. Both the number of appeal stages and the type of appeal which can be held at each stage of appeal are configured on the product.

Example

An organization defines an appeals process for an unemployment insurance product. Three levels are assigned to the product. A hearing review case is designated as the first level of appeal, a judicial review case as the second level of appeal and 'any' type of appeal as the final appeal level.
