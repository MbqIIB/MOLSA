Database Table: IssueDelivery

Table Description
IssueDelivery: A issue delivery is used to store specific data regarding the issue. All other data in relation to the issue can be stored on the caseHeader entity.

Issues can be used to record and manage information in relation to discrepancies in evidence which arise during case processing. They will be created as a new case type and as such can use the majority of existing entities to store data relating to the issue and use existing case management functionality to manage it.
