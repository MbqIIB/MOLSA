Database Table: VerificationAttachmentLink

Table Description
VerificationAttachmentLink : Records an attachment to the provision of a verification item.

When an item is added to a verification, in order to verify a piece of data, this item can be delivered in a number of different forms; a document, a phone call, an email. If the item is electronic, the user can 'attach' the item to the item provision and it can be viewed an maintained at any stage. Alternatively, the user may scan a hard copy document and attach the resulting file. If a hard copy of the item is retained, the user can specify the location and reference of the file as part of the attachment.
