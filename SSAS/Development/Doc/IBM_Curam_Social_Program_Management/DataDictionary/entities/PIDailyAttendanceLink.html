Database Table: PIDailyAttendanceLink

Table Description
PI Daily Attendance Link: A relationship between a planned item and daily attendance.

This entity stores the association between planned items and daily attendances within Service Plan Delivery client participation.

Example

For example, a child was scheduled to receive 8 units of Child Care everyday for the period 01/05/2008-08/05/2008 but he/she received 7 units of it on 07/05/2008. When these daily attendance details are recorded, a link record is also created storing the association between the planned item that the client is participating in and the daily attendance record.
