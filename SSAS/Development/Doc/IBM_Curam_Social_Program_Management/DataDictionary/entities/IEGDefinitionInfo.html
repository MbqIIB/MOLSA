Database Table: IEGDefinitionInfo

Table Description
IEG Object Definition Information : Metadata that allows IEG Object Definitions (Scripts or Question Groups) to be categorized.

There are two types of IEG Object Definitions: Scripts and Question Groups. IEG Object Definition Information records allow domain categories to be defined for these objects. Categorizing IEG Objects in this way restricts access to them - they are not made generally available for editing in the IEG Administration Application.

Example

  • Your application is deployed with a set of generic IEG Objects (ones that do not have IEG Object Definition Information records) and a set of Objects categorized as 'Cúram Decision Assist' (these do have IEG Object Definition Information records).
  • User Ronan Rooney logs in to your application to edit an IEG script. He opens the IEG Administration Application. The only scripts he will see available for editing are the generic IEG Objects (Scripts and Question Groups). He will not see the 'Cúram Decision Assist' objects.

Technical Notes

See the Cúram Server Developers Guide for further information.
