Database Table: CaseDecision

Table Description
Case Decision: The result of the execution of the rules engine against a set of case evidence.

Decisions can be initiated manually by a user, or automatically by the system.

This table holds a type of case event. Events are displayed in an events calendar for viewing by the user.
