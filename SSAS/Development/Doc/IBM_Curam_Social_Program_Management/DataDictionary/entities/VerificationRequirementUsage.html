Database Table: VerificationRequirementUsage

Table Description
Verification Requirement Usage: Records the usage of a particular verification requirement.

The Curam Verification Engine allows an individual verification requirement to be used by many different types of cases.

Example

James Smith's income amount is captured at the integrated case level for a IV-A case. There is a requirement to verify James' income amount. This verification requirement is 'used by' the IV-A integrated Case and a TANF product delivery, as defined in the Curam Administration component.

A TANF product delivery and a Food Stamps product delivery are added to the IV-A integrated case. When a user views the list of verifications for the TANF product delivery, a reference to the income verification appears in the list. However, this verification does not appear in the list of verifications required by the Food Stamps product delivery.
