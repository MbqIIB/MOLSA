Database Table: ContactLogLink

Table Description
Contact Log Link: The link between a Contact Log and any type of case or participant is recorded here.

When a Contact Log is created it is generally associated with a case or participant. The identifier for the linked entity is stored here along with the Contact Log identifier. This is used to record the link between the two entities.

Example

An Investigation case that includes a Contact Log.
