Database Table: AppealSecurity

Table Description
Appeal Security: The security rights definition for all types of appeal cases.

Rights based security provides the ability to check if the user performing an operation has the appropriate level of security based on the security identifier (SID) for that operation. There are a number of fields on the entity to record the SIDs for the following rights:

  • read
  • approve
  • create
  • maintain
  • approve decision

Example

The 'View Details on an Appeal Case' operation has 'Read' security rights associated with it.
