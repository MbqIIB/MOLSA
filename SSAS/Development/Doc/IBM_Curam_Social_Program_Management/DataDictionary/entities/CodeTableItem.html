Database Table: CodeTableItem

Table Description
Code Table Item : A translation of a code into a language-specific textual description.

Each Code Table Item can specify its relative sort order within all other codes belonging to the same Code Table Header. This sorting controls the order in which the code table's descriptions appear in the Cúram client user interface. If more than one item in a Code Table Header shares the same sort order value, this group of items will in alphabetical order.

Each Code Table Item can be enabled or disabled by an administrator.

Example

User Ronan Rooney opens a screen which instructs him to choose a weekday from a list. The screen presents the list Monday, Tuesday, Wednesday, Thursday and Friday in their natural ordering (not in alphabetical order).

An administrator adds the 'Euro' currency to a currency code table, and at some later stage disables the old code table values 'Lira', 'Francs' and 'Deutschemarks'. User Ronan Rooney opens a screen which shows old financial records, which correctly display records in Lira, Francs and Deutschemarks; however, when he adds new financial records Euro is present in the list of available currencies but Lira, Francs and Deutschemarks are not.

Technical Notes

If you disable a Code Table Item, you should ensure that the decodes for all your supported languages are disabled.

For more information see the Cúram Server Developer's Guide.
