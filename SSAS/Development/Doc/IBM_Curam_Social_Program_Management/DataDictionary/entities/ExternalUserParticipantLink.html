Database Table: ExternalUserParticipantLink

Table Description
External User Participant Link: This entity links an external user to participants with whom they have a relationship.

Example

The relationship might be the external user account profile for the participant itself, for example an employer linked to their own participant account. Or another relationship might be to other participant records on the system that the external user is allowed access, for example an employer allowed to access their employee participant data.
