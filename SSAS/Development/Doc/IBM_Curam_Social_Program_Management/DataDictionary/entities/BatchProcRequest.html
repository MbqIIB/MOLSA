Database Table: BatchProcRequest

Table Description
Batch Process Request: A request for a batch process to be executed.

Example

User Ronan Rooney requests that the batch process MyBatchClass.myBatchMethod should be executed. The Cúram Server stores a row on this table to record the request.

At some later stage (perhaps after the online day), a batch operator starts the Cúram Batch Launcher, which:

  • invokes the batch process MyBatchClass.myBatchMethod which User Ronan Rooney requested; and
  • after successful invocation, removes the row from this table.

Technical Notes

For more information on the Cúram Batch Launcher, see the Cúram Operations Guide and the Cúram Batch Processing Guide.
