Database Table: TaskHistory

Table Description
Task History : An event in the lifecycle of a Task.

A row on this table stores the old and new values for a particular column value which has changed on the related Task row.

Example

A Task is assigned to User Ronan Rooney to instruct him to post a letter to Person John Smith.

Ronan reserves the Task to start work on it. Curam changes the reservedBy value on the Task to be User Ronan Rooney and creates a new row on the Task History to record that the reservedBy value has changed.

Ronan is unable to complete the work before going on holiday and so:

  • un-reserves the Task - Curam clears the reservedBy value on the Task and creates a new corresponding row on the Task History table; and
  • reassigns the Task to his supervisor - Curam sets the assignedDateTime on the Task and creates a new corresponding row on the Task History table.
