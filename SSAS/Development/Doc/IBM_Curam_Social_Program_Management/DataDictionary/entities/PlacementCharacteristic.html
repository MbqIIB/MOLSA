Database Table: PlacementCharacteristic

Table Description
Placement Characteristic: Definition of characteristics for a provider which are used to identify appropriate placements for clients within compartments.

Definition of characteristics for a provider which are used to identify appropriate placements for clients within compartments.

Example

A Care home for ladies aged 50 and above is set up with Minimum Age of 50 and a Gender of 'Female Only'.
