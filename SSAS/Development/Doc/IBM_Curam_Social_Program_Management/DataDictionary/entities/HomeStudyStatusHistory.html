Database Table: HomeStudyStatusHistory

Table Description
Home Study Status History: The status history of a home study.

The status of a home study will change through its lifecycle, for example, the status for a home study is set to 'Open' upon initial creation, and then changed to 'Submitted' once submitted.

Example

The status for a home study is set to 'Open' upon initial creation, and then changed to 'Submitted' once submitted.
