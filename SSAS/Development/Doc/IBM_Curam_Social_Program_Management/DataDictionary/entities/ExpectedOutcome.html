Database Table: ExpectedOutcome

Table Description
Expected Outcome : An objective that can be be added to an outcome plan for one or more clients. An expected outcome must be associated with a type of outcome plan in adminsitration before it can be selected for a client on that outcome plan.

Example

A case worker assesses James Smith's strengths and needs and identifies that he has problems finding work and is alcohol dependent. To address these issues the case worker creates an outcome plan. A plan goal of self sufficiency is added to the plan. An objective of 'Remain Clean and Sober' is selected and added to the plan. This entity allows the objective 'Remain Clean and Sober' to be selected when adding an objective to an outcome plan.
