Database Table: PhoneNumber

Table Description
Phone Number: A telephone number.

Technical Notes

Many types of Curam entities can have a related phone number; logically these entities have a 'phone number' attribute which is a 'complex data type'. Physically however each of these entities has a phone number attribute which relates to a row on the phone number table.
