Database Table: RateHeader

Table Description
Rate Header: An effective date from which rates in the rate table have changed.

The rates are defined in the rate columns and rate cells which related to this rate header. These rates apply up to the date of the next rate header for the same rateTableType (if such a 'next' rate header exists).

For more information on rate tables, see the Curam Application Administration guide.
