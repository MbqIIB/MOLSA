Database Table: ProgramAuthorisationData

Table Description
ProgramAuthorisationData : A Program Authorisation Data record is used to store the details of a program authorization on the application case. This entity is used to store information on the program authorization action which is then available for user review.

Example

James Smith has a Social Benefits application case for Food Allowance and Rent Allowance. A worker processing the application determines that James is eligible for Food Allowance and and authorizes the program on the application case. The Food Allowance program status is updated to Authorized, and a ProgramAuthorizationData record is created.
