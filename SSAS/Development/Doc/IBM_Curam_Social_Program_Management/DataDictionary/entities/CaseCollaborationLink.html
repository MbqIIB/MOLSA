Database Table: CaseCollaborationLink

Table Description
Case Collaboration Link : A link entity that creates a one to one relationship between a case and a Social Enterprise Folder. This entity is used to link a case to an Social Enterprise Folder so that a Multidisciplinary Team can collaborate on the case.

Technical Notes

Optimistic locking is not switched on for this entity as no modify operations exist A case can only be linked to one Social Enterprise Folder and visa versa.
