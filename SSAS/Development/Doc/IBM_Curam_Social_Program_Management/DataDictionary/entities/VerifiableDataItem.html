Database Table: VerifiableDataItem

Table Description
Verifiable Data Item: Identifies a data item that requires verification.

This entity is used to identify the evidence data items that requirement verification.

An SEM agency has a requirement to verify a client's income for a particular benefit. Therefore a verifiable data item must be created to represent the evidence data item that is used to capture the client's income. The evidence data item is identified by a combination of the evidence descriptor type and the attribute name.
