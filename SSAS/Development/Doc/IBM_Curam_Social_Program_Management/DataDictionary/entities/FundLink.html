Database Table: FundLink

Table Description
: A prioritized link between a fund and a product or service. This table allows an organization to create a link between funds and the products or services which are authorized to use the fund's budgets. It also allows the organization to specify a priority by which the funding streams should be spent. The organization creates three FundLink records to associate a Welfare benefit product to Federal, State and Municipal funds. These records are created with a priority of (1)Federal, (2) Municipal and (3) State in order for the organization to specify that Federal funds should be utilized first.
