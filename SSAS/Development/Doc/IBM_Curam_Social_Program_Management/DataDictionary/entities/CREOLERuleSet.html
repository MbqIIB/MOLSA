Database Table: CREOLERuleSet

Table Description
Rule Set : A rule set for Cúram Express Rules.

A set of rules complying with the CREOLE schema and validation constraints.

The full XML definition of the rule set is held in the ruleSetDefinition column, but the Cúram application will denormalize certain structural details of the rule set to these entities:

  • Rule Set Dependency;
  • Rule Class;
  • Rule Class Inheritance;
  • Rule Attribute;
  • Rule Attribute Availability; and
  • Rule Attribute Inheritance.

Technical Notes

Other than uploading data via DMX files, the data on this table may only be read or written using CREOLE APIs. Rule sets uploaded via DMX data will have their structural details denormalized by the Cúram application when it first accesses the uploaded rule set.

The following constraints apply to rule sets uploaded via DMX files:

  • the value for the ruleSetVersion column must be NULL;
  • the value for the migrationAction column must be NULL; and
  • the value for the name column must match the rule set name declared in the XML for the ruleSetDefinition column;
