Database Table: QuestionOutcomeMetaData

Table Description
DAQuestion Decision Outcome MetaData: Questionnaire Decision Outcome Meta Data entity stores presentation code for which Questionnaire Versions are presented. This may be for certain Determination Outcomesor general presentation data for the Questionnaire Versions

Questionnaire Decision Outcome Meta Data stores information to determination which questionnaires are displayed and how they are displayed. This may be for certain Decision Matrix Outcomes or general presentation data for the Questionnaire Version.
