Database Table: IssueConfiguration

Table Description
Issue Configuration: Configuration details of a type of issue which can arise during case processing.

This table stores the configuration details of types of issues which can be created on a product delivery or integrated case.

Example

During case processing a discrepancy is identified between the earned income details provided by the client and those provided by the client's employer. An earned income issue is automatically created. To facilitate this, an issue of type Earned Income must be created in the Administration component. The configuration details of the Earned Income issue are stored on the Issue Configuration table.
