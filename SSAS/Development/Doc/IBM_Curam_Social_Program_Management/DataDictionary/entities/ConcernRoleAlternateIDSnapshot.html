Database Table: ConcernRoleAlternateIDSnapshot

Table Description
Concern Role Alternate ID Snapshot: Maintains a history of changes to concern role alternateID record.

Example

If a user modifies, for example, comments on a concern role alternateID record, a snapshot of the concern role alternateID data prior to making this modification is written to the concern role alternateID snapshot table. Likewise when any of the concern role alternateID details are modified a snapshot of the previous version of the record is created.
