Database Table: Attachment

Table Description
Attachment: A file that can be associated with a communication or a case.

Example

User Ronan Rooney attaches a Microsoft Word document a case.

Technical Notes

An attachment is stored as a binary large object (BLOB) in the database. Any file format can be stored as an attachment.

Many types of Curam entities can have a related attachment; logically these entities have an 'attachment' attribute which is a 'complex data type'. Physically however each of these entities has an attachment attribute which relates to a row on the attachment table.
