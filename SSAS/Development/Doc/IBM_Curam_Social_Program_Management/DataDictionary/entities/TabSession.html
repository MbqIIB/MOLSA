Database Table: TabSession

Table Description
Tab Session : A record of the tab session data for a user.

Tab Session stores a record of the current tabs open for a particular user. This data is used to restore a users session after both a browser refresh (f5) and after logging out of the application. The data may be fragmented across multiple records if it grows too big. It will be reassembled according to the sequence number of the fragments.

Example

User Ronan Rooney logs in to the application and opens a number of tabs, including the Person tabs for James Smith and Linda Smith. He the logs out of the application and subsequently logs in from a different machine. After login, the previously displayed tabs for James Smith and Linda Smith are still open in his view.

Technical Notes

For more information on tab session management see the Cúram Web Client Reference Manual.
