Database Table: DeterminationConfig

Table Description
DADetermination Configuration: The Determination Configuration entity stores details of the Determination Configurations available in the system and is the means by which a Determination Configuration is set up.

A Determination Configuration is the means by which a determination is configured in decision assist administration. Instances of these Determinations Configurations are used in Decision Assist Case Manager and associated with Assessment Deliveries.
