Database Table: EvidenceBrokerConfig

Table Description
EvidenceBrokerConfig: Stores the configuration of the evidence to be shared via Evidence Broker.

The EvidenceBrokerConfig entity captures the configuration of the evidence to be shared via Evidence Broker.
