Database Table: ProcessDefinitionTranslation

Table Description
Process Definition Translation : Contains localized names for the associated Process Definition.

This entity contains the localized process display name for the associated Process Definition.

Example

You define a process using the Process Definition Tool (PDT). Curam creates a row on this table to store the display name of your Process Definition.

Technical Notes

This process display name is always editable unlike the process name (which is the identifier).

The display name must be unique within all the processes in the system.
