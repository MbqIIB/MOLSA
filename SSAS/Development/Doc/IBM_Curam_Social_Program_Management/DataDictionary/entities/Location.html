Database Table: Location

Table Description
Location: A physical workplace for an organization's employees.

The organization's locations are organized in a hierarchical location relationship structure having one root location.

Example

An organization in a State operates a multi-level location hierarchy:

At the top of the hierarchy is the root location. Within this location, there may be many child locations. These child locations may have further child locations.
