Database Table: DAOutcome

Table Description
DADAOutcome: An outcome is the result that can be obtained from a particular decision matrix.

An outcome is a result that can be obtained from a particular decision matrix. The combination of outcome and decision matrix type and score where appropriate are what constitutes a determination result.
