Database Table: MergedRecord

Table Description
Merged Record: A list of new records created as a result of a merge.

Details of any information added to the original client as a result of a merge.

Example

Jimmy Smith has been marked as a duplicate of James Smith. The user selects to merge selected details about Jimmy Smith (the duplicate) to James Smith (the master record). A history is kept of the records that were merged and the new records that were created as a result of the merge process.
