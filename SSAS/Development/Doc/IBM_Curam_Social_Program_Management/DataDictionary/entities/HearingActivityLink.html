Database Table: HearingActivityLink

Table Description
Hearing Activity Link: The relationship between a hearing and the activities created for that hearing.

When user appeals for an appeal case a hearing scheduler schedules a hearing to review the case by hearing officials. Then system inserts an activity in the hearing official's calendar for a particular date, time and location.

Example

Linda Smith applies for sickness benefit with an organization. It is calculated that Linda is entitled to $40 a week. Linda is dissatisfied with this rate of payment and submits a request for appeal to the organization.

A user creates an appeal case in the system. A hearing scheduler schedules a hearing where Linda's case will be reviewed by a hearing official. The system inserts an activity in the hearing official's calendar for a particular date, time and location.
