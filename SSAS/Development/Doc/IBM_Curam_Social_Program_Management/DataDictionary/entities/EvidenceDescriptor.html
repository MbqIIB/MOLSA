Database Table: EvidenceDescriptor

Table Description
Evidence Descriptor: The evidence descriptor contains information standard across all evidence types.

The role of the evidence descriptor entity is similar to that of the case header. Where the case header entity contains the information common across case types, evidence descriptor contains information common across all evidence types. This entity allows common evidence information to be stored in a single database table, which supports centralizing and standardizing of evidence maintenance functions.

Example

An example of common evidence data is the status of an evidence record or the date on which evidence is received by the organization. These data items are required regardless of whether capturing income evidence, loan evidence etc. An example of centralizing evidence maintenance functions is that common evidence information allows us provide a summary of evidence, therefore supporting a common evidence workspace and evidence approval process etc.
