Database Table: AppealStageConfiguration

Table Description
Appeal Stage Configuration: A level within an appeal process.

An appeal process is made up of one or more appeal stages. An appeal stage can be defined as a specific type of appeal case. This means that only an appeal case of that type can be created for that stage of appeal. Alternatively, a generic value of Any can be selected which means that any type of appeal case can be created for this stage of appeal.

Example

An organization defines an appeal process for an issue on an income support product and defines the first stage of appeal to be a hearing case, the second stage of appeal to be also a hearing case and the third level of appeal to be 'any' type of appeal.
