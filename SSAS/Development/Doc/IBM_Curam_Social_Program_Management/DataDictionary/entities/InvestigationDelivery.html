Database Table: InvestigationDelivery

Table Description
Investigation Delivery: An inquiry into the circumstances surrounding an allegation or incident.

This table stores information specific to an investigation which is not stored on the case header table.

Example

Examples of an investigation are Child Abuse, Child Neglect, Benefit Fraud.
