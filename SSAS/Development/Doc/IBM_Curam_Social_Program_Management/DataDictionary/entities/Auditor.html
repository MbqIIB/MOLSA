Database Table: Auditor

Table Description
Auditor : Details relating to an auditor on an audit plan.

This entity captures details relating to an auditor on an audit plan. An auditor is the user responsible for examining and evaluating particular case audit records assigned to them.

Example

An audit coordinator creates an audit plan and generates a list of cases to audit. These audits need to be assigned out for examination. The audit coordinator adds a new auditor by searching for a particular user and adds them to the audit plan. Case Audits can now be assigned to this auditor.
