Database Table: InsuranceReturnLineItem

Table Description
Insurance Return Line Item: A unit attributed to a person which is returned by an employer for a period over which that person was employed by the employer.

These details are used in the calculation of Entitlement to Products in groups. Each InsuranceReturnLineItem has its respective details in the corresponding InsuranceRetLineItemDetail entity record.
