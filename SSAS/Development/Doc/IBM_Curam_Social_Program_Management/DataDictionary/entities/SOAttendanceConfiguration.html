Database Table: SOAttendanceConfiguration

Table Description
Service Offering Attendance Configuration: The attendance tracking configuration settings for a service offering.

The attendance tracking configuration settings for a service offering.

Example

Carry's Child Care is a registered provider for providing child care services to the organization's clients. Ronan Smith is authorized by the organization to receive child care from 9 A.M. to 5 P.M. every day from Carry's Child Care. The organization has administratively set the Attendance Tracking Required and Daily Attendance Tracking Required Indicator to true for the Child Care service offering. The Roster Generation Frequency is set to current date. Hence a roster will be generated weekly as the Attendance Tracking Required indicator is set to true and frequency is set as weekly for service for the current period.
