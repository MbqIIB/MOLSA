Database Table: CodeTableData

Table Description

Code Table Data: A table to store comments for Code Table Header.

Code Table Data can store can store comments for Code Table Header in different locales. The comments field can store upto 4000 characters.

Example

The comments field can be used to store any documentation for the users to understand the internal.codetable.CodeTableHeader Code Table Header better. This documentation can also be used to know if there are any implications in changing this code table.
