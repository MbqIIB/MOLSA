Database Table: AuditTeam

Table Description
Audit Team : A team of auditors on an audit plan.

An audit team represents one or more auditors responsible for jointly examining and evaluating particular case records.

Example

Auditors can be grouped into an audit team. An audit team represents one or more auditors responsible for jointly examining and evaluating particular case records. Once a case list has been generated they can then be assigned to an audit team for examination. Any auditor within the team can have input on the case audit.
