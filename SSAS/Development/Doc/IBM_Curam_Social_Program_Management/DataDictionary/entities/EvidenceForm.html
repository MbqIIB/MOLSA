Database Table: EvidenceForm

Table Description
Evidence Form : Represents different evidence forms for a product. Represents different evidence forms for a product.

Technical Notes

Please Note: This entity is no longer part of the default flow in the Cúram Application, as such it is neither written to nor read from by the Cúram Application. It continues to be shipped with the Cúram model as customer may have custom logic which uses this entity, however it should be noted that if you are implementing custom logic which references this entity then you must also ensure that it is written to as needed. The information that was originally stored in this entity is now stored in evidence screen.
