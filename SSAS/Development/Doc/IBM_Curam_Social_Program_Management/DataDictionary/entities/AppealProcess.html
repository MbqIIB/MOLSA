Database Table: AppealProcess

Table Description
Appeal Process: The appeal process definition for any type of case.

An appeal process is composed of one or more appeal stages. In addition, a single case can have one or more appeal processes, each with a different start date. Both the number of appeal stages and the type of appeal which can be held at each stage of appeal case be configured.

Example

An organization defines an appeals process for an issue which has arisen on an unemployment insurance product. Three levels are assigned to the issue. A hearing review case is designated as the first level of appeal, a judicial review case as the second level of appeal and 'any' type of appeal as the final appeal level.
