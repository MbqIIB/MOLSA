Database Table: ProductDeliveryPlanItemLink

Table Description
Product Delivery Plan Item Link: A link between a plan item and a product delivery case.

This entity is focused on the relationship between plan items and product delivery cases during the delivery of a plan item of this type. The service plan administration component provides a framework to allow integration between plan items and other Curam objects, in this case a Product delivery case. When a plan item of this type is created, this entity is updated with the details of the link between the plan item and the product delivery case.

Example

A case worker creates a Return to Work service plan for participant Linda Smith as she has recently become unemployed. A goal of Sustainable Employment is selected for this service plan. Linda requires financial assistance to subsidize her travel to work, therefore a Transportation sub-goal is added to the service plan. A Mileage Reimbursement Plan Item is added to the service plan to achieve the service plan goal.

The Mileage Reimbursement plan item is submitted for approval. A Mileage Reimbursement product delivery case can be created at this stage.

This link entity enables the integration between a many Curam objects and a service plan item.
