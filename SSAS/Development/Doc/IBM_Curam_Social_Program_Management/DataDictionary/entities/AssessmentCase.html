Database Table: AssessmentCase

Table Description
Assessment Case : An assessment of a client across one or more defined factors.

An assessment case record has an associated case header record of type Assessment Case. Assessments can also be associated with any case header type e.g. integrated case, outcome plan.

Example

James Smith, Linda Smith and their child Robert must be assessed to identify the family's strengths and needs in an effort to help build an effective outcome plan. The Definition Family Strengths and Needs Assessment (FSN) is run from the family's outcome plan by a case worker. Linda is selected as Roberts primary caregiver, James is his secondary caregiver and Robert is selected as the focus child. The Definition FSN contains nine factors which are used to assess the family's functioning and a result is produced for each factor. This FSN assessment can be reassessed to assess the family's progress over time. This entity is an extension of the case header record and is used to associate an assessment with the case header that the assessment is being run from.
