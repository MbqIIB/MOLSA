Database SQL Operation: SPMilestoneDeliveryLink.searchMilestonesAndLinksByCaseIDAndPlannedGroupID

Operation Details
Description: Method to retrieve a list of milestones created at planned group level and their service plan link details.
Sterotype: nsmulti

SQL 
SELECT
  MilestoneDelivery.milestoneDeliveryID,
  MilestoneDelivery.milestoneConfigurationID,
  MilestoneDelivery.caseID,
  MilestoneDelivery.ownerUserName,
  MilestoneDelivery.expectedStartDate,
  MilestoneDelivery.expectedEndDate,
  MilestoneDelivery.actualStartDate,
  MilestoneDelivery.actualEndDate,
  MilestoneDelivery.comments,
  MilestoneDelivery.status,
  SPMilestoneDeliveryLink.spMilestoneDeliveryLinkID,
  SPMilestoneDeliveryLink.plannedGroupID,
  SPMilestoneDeliveryLink.plannedSubGoalID
INTO
  :milestoneDeliveryID,
  :milestoneConfigurationID,
  :caseID,
  :ownerUserName,
  :expectedStartDate,
  :expectedEndDate,
  :actualStartDate,
  :actualEndDate,
  :comments,
  :status,
  :spMilestoneDeliveryLinkID,
  :plannedGroupID,
  :plannedSubGoalID 
FROM
  MilestoneDelivery,
  SPMilestoneDeliveryLink 
WHERE MilestoneDelivery.caseID = :caseID 
  AND SPMilestoneDeliveryLink.milestoneDeliveryID = MilestoneDelivery.milestoneDeliveryID 
  AND SPMilestoneDeliveryLink.plannedGroupID = :plannedGroupID
