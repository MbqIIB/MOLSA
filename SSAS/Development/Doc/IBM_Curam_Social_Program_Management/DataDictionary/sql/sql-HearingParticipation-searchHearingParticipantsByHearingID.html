Database SQL Operation: HearingParticipation.searchHearingParticipantsByHearingID

Operation Details
Description: Lists all the legal participants for a hearing
Sterotype: nsmulti

SQL 
SELECT
  HearingParticipation.hearingParticipationID,
  HearingParticipation.caseParticipantRoleID,
  CaseParticipantRole.participantRoleID,
  HearingParticipation.versionNo,
  ConcernRole.concernRoleName,
  HearingParticipation.participatedCode,
  CaseParticipantRole.typeCode
INTO
  :hearingParticipationID,
  :caseParticipantRoleID,
  :participantRoleID,
  :versionNo,
  :fullName,
  :participatedCode,
  :participantType 
FROM
  HearingParticipation,
  CaseParticipantRole,
  ConcernRole 
WHERE HearingParticipation.hearingID = :hearingID 
  AND CaseParticipantRole.recordStatus = :recordStatus 
  AND HearingParticipation.caseParticipantRoleID = CaseParticipantRole.caseParticipantRoleID 
  AND CaseParticipantRole.participantRoleID = ConcernRole.concernRoleID
