Database SQL Operation: PDApprovalRequest.searchByCaseID

Operation Details
Description:  
Sterotype: nsmulti

SQL 
SELECT
  ApprovalRequest.approvalRequestID,
  ApprovalRequest.requestedDate,
  ApprovalRequest.requestedByUser,
  ApprovalRequest.approvalDecisionDate,
  ApprovalRequest.approvalDecisionByUser,
  ApprovalRequest.documentRefNo,
  ApprovalRequest.approverTypeCode,
  ApprovalRequest.methodCode,
  ApprovalRequest.automaticallyApprovedInd,
  ApprovalRequest.status,
  ApprovalRequest.rejectionReason,
  ApprovalRequest.rejectionComments
INTO
  :approvalRequestID,
  :requestedDate,
  :requestedByUser,
  :approvalDecisionDate,
  :approvalDecisionByUser,
  :documentRefNo,
  :approverTypeCode,
  :methodCode,
  :automaticallyApprovedInd,
  :status,
  :rejectionReason,
  :rejectionComments 
FROM
  PDApprovalRequest,
  ApprovalRequest 
WHERE PDApprovalRequest.caseID = :caseID 
  AND ApprovalRequest.approvalRequestID = PDApprovalRequest.approvalRequestID
