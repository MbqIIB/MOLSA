Database SQL Operation: DeterminationAnswer.searchDAIDsByDDIDAndQVID

Operation Details
Description: Gets all the answer IDs for the given determination delivery and the questionnaire version.
Sterotype: nsmulti

SQL 
SELECT
    determinationAnswerID INTO :determinationAnswerID
  FROM
    DeterminationAnswer DA1,
    Question Q1 
  WHERE DA1.determinationDeliveryID = :determinationDeliveryID 
    AND Q1.questionnaireVersionID = :questionnaireVersionID 
    AND DA1.questionID = Q1.questionID 
    AND DA1.lastUpdatedDateTime IN 
    ( 
    SELECT
      MAX(lastUpdatedDateTime) 
    FROM
      DeterminationAnswer DA2,
      Question Q2 
    WHERE DA2.determinationDeliveryID = :determinationDeliveryID 
      AND Q2.questionnaireVersionID = :questionnaireVersionID 
      AND DA2.questionID = Q2.questionID 
    GROUP
      BY Q2.questionID 
    )
