Database SQL Operation: PositionSlotAvailability.searchByEffectiveDateAndSchedule

Operation Details
Description: returns the position slot availability id based on an effective date, a status code and a daily schedule id
Sterotype: nsmulti

SQL 
SELECT
  PSA.positionSlotAvailabilityID
INTO
  :positionSlotAvailabilityID 
FROM
  PositionSlotAvailability PSA,
  Slot 
WHERE PSA.fromDate = 
  ( 
  SELECT
    MAX(fromDate) 
  FROM
    PositionSlotAvailability 
  WHERE fromDate <= :fromDate 
  ) 
  AND Slot.scheduleID = :scheduleID 
  AND Slot.slotID = PSA.slotID 
  AND PSA.recordStatus = :recordStatus
