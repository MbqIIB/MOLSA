Database SQL Operation: SPMilestoneDeliveryLink.searchUncompletedMilestonesForServicePlan

Operation Details
Description: @deprecated since curam 6.0, replaced by {@Link SPMilestoneDeliveryLink#searchUncompletedMilestonesForServicePlan1()} See release note:CR00236426 This method builds the default list of milestones for a service plan. This list contains milestones at service plan level with a status of Not Started or In Progress.
Sterotype: nsmulti

SQL 
SELECT
  MilestoneDelivery.milestoneDeliveryID,
  MilestoneConfiguration.name,
  MilestoneConfiguration.type,
  MilestoneDelivery.expectedStartDate,
  MilestoneDelivery.expectedEndDate,
  MilestoneDelivery.actualStartDate,
  MilestoneDelivery.actualEndDate,
  MilestoneDelivery.status
INTO
  :milestoneDeliveryID,
  :name,
  :type,
  :expectedStartDate,
  :expectedEndDate,
  :actualStartDate,
  :actualEndDate,
  :status 
FROM
  MilestoneConfiguration,
  MilestoneDelivery,
  SPMilestoneDeliveryLink 
WHERE SPMilestoneDeliveryLink.milestoneDeliveryID = MilestoneDelivery.milestoneDeliveryID 
  AND MilestoneDelivery.caseID = :caseID 
  AND MilestoneDelivery.status <> :status 
  AND MilestoneDelivery.milestoneConfigurationID = MilestoneConfiguration.milestoneConfigurationID
