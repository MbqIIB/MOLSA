Database SQL Operation: ICAssessmentConfiguration.searchByAdminIntegratedCaseID

Operation Details
Description: Returns a list of assessment configurations, regardless of the status, associated with an integrated case configuration.
Sterotype: nsmulti

SQL 
SELECT
  ICAssessmentConfiguration.icAssessmentConfigurationID,
  ICAssessmentConfiguration.assessmentConfigurationID,
  AssessmentConfiguration.name,
  ICAssessmentConfiguration.startDate,
  ICAssessmentConfiguration.endDate,
  ICAssessmentConfiguration.recordStatus
INTO
  :icAssessmentConfigurationID,
  :assessmentConfigurationID,
  :name,
  :startDate,
  :endDate,
  :recordStatus 
FROM
  ICAssessmentConfiguration,
  AssessmentConfiguration 
WHERE ICAssessmentConfiguration.adminIntegratedCaseID = :adminIntegratedCaseID 
  AND AssessmentConfiguration.assessmentConfigurationID = ICAssessmentConfiguration.assessmentConfigurationID
