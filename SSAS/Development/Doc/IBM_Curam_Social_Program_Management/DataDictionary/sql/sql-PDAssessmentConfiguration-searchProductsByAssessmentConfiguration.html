Database SQL Operation: PDAssessmentConfiguration.searchProductsByAssessmentConfiguration

Operation Details
Description: Returns a list of products, regardless of the status, associated with an assessment configuration.
Sterotype: nsmulti

SQL 
SELECT
  Product.productID,
  Product.name,
  Product.typeCode,
  PDAssessmentConfiguration.startDate,
  PDAssessmentConfiguration.endDate,
  Product.statusCode
INTO
  :productID,
  :productName,
  :productType,
  :startDate,
  :endDate,
  :statusCode 
FROM
  Product,
  PDAssessmentConfiguration 
WHERE PDAssessmentConfiguration.assessmentConfigurationID = :assessmentConfigurationID 
  AND Product.productID = PDAssessmentConfiguration.productID
