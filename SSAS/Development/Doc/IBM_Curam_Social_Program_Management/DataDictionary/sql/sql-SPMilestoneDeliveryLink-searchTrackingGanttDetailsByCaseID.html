Database SQL Operation: SPMilestoneDeliveryLink.searchTrackingGanttDetailsByCaseID

Operation Details
Description: @deprecated since curam 6.0, replaced by {@Link SPMilestoneDeliveryLink#searchTrackingGanttDetailsByCaseID1()} See release note:CR00236426 Method to search milestones created at service plan level for display on tracking gantt.
Sterotype: nsmulti

SQL 
SELECT
  MilestoneDelivery.milestoneDeliveryID,
  MilestoneConfiguration.name,
  MilestoneDelivery.expectedStartDate,
  MilestoneDelivery.expectedEndDate,
  MilestoneDelivery.actualStartDate,
  MilestoneDelivery.actualEndDate
INTO
  :milestoneDeliveryID,
  :name,
  :expectedStartDate,
  :expectedEndDate,
  :actualStartDate,
  :actualEndDate 
FROM
  MilestoneConfiguration,
  MilestoneDelivery,
  SPMilestoneDeliveryLink 
WHERE SPMilestoneDeliveryLink.milestoneDeliveryID = MilestoneDelivery.milestoneDeliveryID 
  AND MilestoneDelivery.caseID = :caseID 
  AND MilestoneDelivery.milestoneConfigurationID = MilestoneConfiguration.milestoneConfigurationID 
  AND SPMilestoneDeliveryLink.plannedGroupID is NULL 
  AND SPMilestoneDeliveryLink.plannedSubGoalID is NULL
