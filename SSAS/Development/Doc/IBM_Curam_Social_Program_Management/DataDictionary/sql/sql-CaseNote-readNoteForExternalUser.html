Database SQL Operation: CaseNote.readNoteForExternalUser

Operation Details
Description: @deprecated - replaced by {@link #readNoteForExternalUser1()} @deprecated-since Version 6.0
Sterotype: ns

SQL 
SELECT
  CaseNote.caseNoteID,
  CaseNote.caseID,
  CaseNote.recordStatus,
  Note.sensitivityCode,
  Note.notesText,
  Note.creationDateTime,
  Note.priorityCode,
  ExternalUser.fullName,
  Note.userName,
  CaseNote.noteID,
  CaseNote.versionNo
INTO
  :caseNoteID,
  :caseID,
  :recordStatus,
  :sensitivityCode,
  :notesText,
  :creationDateTime,
  :priorityCode,
  :fullname,
  :userName,
  :noteID,
  :versionNo 
FROM
  CaseNote,
  Note,
  ExternalUser 
WHERE CaseNote.caseNoteID = :caseNoteID 
  AND Note.noteID = CaseNote.noteID 
  AND Note.userName = ExternalUser.userName
