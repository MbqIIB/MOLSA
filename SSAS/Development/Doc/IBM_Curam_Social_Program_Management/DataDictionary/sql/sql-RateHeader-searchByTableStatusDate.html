Database SQL Operation: RateHeader.searchByTableStatusDate

Operation Details
Description: Searches rate headers based on table type, rate status code and effectiveDate specified.
Sterotype: nsmulti

SQL 
SELECT
  rateHeaderID,
  effectiveDate,
  rateTableType,
  comments,
  rateStatus,
  versionNo
INTO
  :rateHeaderID,
  :effectiveDate,
  :rateTableType,
  :comments,
  :rateStatus,
  :versionNo 
FROM
  RateHeader 
WHERE 
  (
    
    (
      :rateTableType IS NULL 
    ) 
    OR rateTableType = :rateTableType 
  ) 
  AND 
  (
    
    (
      :rateStatus IS NULL 
    ) 
    OR rateStatus = :rateStatus 
  ) 
  AND 
  (
    
    (
      :effectiveDate IS NULL 
    ) 
    OR effectiveDate = :effectiveDate 
  )
