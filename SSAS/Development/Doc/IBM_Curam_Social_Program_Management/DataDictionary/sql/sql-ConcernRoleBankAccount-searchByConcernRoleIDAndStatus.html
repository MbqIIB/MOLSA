Database SQL Operation: ConcernRoleBankAccount.searchByConcernRoleIDAndStatus

Operation Details
Description: Returns a list of Concern Role Bank Account records by concernRoleID and status.
Sterotype: nsmulti

SQL 
SELECT
  BankAccount.name,
  BankAccount.typeCode,
  BankAccount.bankSortCode,
  BankAccount.bankBranchID,
  BankAccount.accountNumber,
  BankAccount.statusCode,
  ConcernRoleBankAccount.concernRoleBankAccountID,
  BankAccount.bankAccountID,
  BankAccount.versionNo
INTO
  :name,
  :typeCode,
  :bankSortCode,
  :bankBranchID,
  :accountNumber,
  :statusCode,
  :concernRoleBankAccountID,
  :bankAccountID,
  :versionNo 
FROM
  BankAccount,
  ConcernRoleBankAccount 
WHERE ConcernRoleBankAccount.concernRoleID = :concernRoleID 
  AND BankAccount.statusCode = :statusCode 
  AND BankAccount.bankAccountID = ConcernRoleBankAccount.bankAccountID
