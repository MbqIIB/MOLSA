Database SQL Operation: QuestionnaireVersion.searchQuestionnaireForPresentationData

Operation Details
Description: Reads details of most recent versions of all questionnaires associated with a specified version of a determination package
Sterotype: nsmulti

SQL 
SELECT
  QV1.questionnaireVersionID,
  QV1.questionnaireID,
  QV1.name,
  QV1.type,
  QV1.questionnaireVersionNumber,
  QV1.versionStatus,
  QV1.fromDate,
  QV1.toDate,
  DPQ1.dpQuestionnaireLinkID
INTO
  :questionnaireVersionID,
  :questionnaireID,
  :name,
  :type,
  :version,
  :status,
  :fromDate,
  :toDate,
  :dpQuestionnaireLinkID 
FROM
  QuestionnaireVersion QV1,
  DPQuestionnaireLink DPQ1 
WHERE QV1.questionnaireVersionID = DPQ1.questionnaireVersionID 
  AND DPQ1.determinationPackageVersionID = :determinationPackageVersionID 
  AND QV1.questionnaireVersionNumber = 
  ( 
  SELECT
    max(questionnaireVersionNumber) 
  FROM
    QuestionnaireVersion QV2,
    DPQuestionnaireLink DPQ2 
  WHERE QV2.questionnaireVersionID = DPQ2.questionnaireVersionID 
    AND DPQ1.determinationPackageVersionID = DPQ2.determinationPackageVersionID 
    AND QV1.questionnaireID = QV2.questionnaireID 
  ) 
ORDER BY QV1.name
