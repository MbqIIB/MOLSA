Database SQL Operation: CaseUserRole.readActiveUserDetails

Operation Details
Description: Returns userName and caseUserRoleID based on caseID, statusCode and typeCode. this method returns a single record (i.e. when typeCode is SUPERVISOR or OWNER)
Sterotype: ns

SQL 
SELECT
  CaseUserRole.orgObjectLinkID,
  CaseUserRole.caseUserRoleID,
  OrgObjectLink.userName,
  OrgObjectLink.orgObjectReference,
  CaseUserRole.typeCode
INTO
  :orgObjectLinkID,
  :caseUserRoleID,
  :userName,
  :orgObjectRef,
  :caseUserRoleType 
FROM
  CaseUserRole,
  OrgObjectLink 
WHERE CaseUserRole.caseID = :caseID 
  AND CaseUserRole.typeCode = :typeCode 
  AND CaseUserRole.recordStatus = :recordStatus 
  AND OrgObjectLink.orgObjectLinkID = CaseUserRole.orgObjectLinkID
