Database SQL Operation: OrgObjectLink.readUserNameByCaseIDTypeStatus

Operation Details
Description: Reads the user name for a case. This is primarily used in returning the userName for the case supervisor and the case reviewer
Sterotype: ns

SQL 
SELECT
  OrgObjectLink.userName
INTO
  :userName 
FROM
  OrgObjectLink,
  CaseUserRole 
WHERE CaseUserRole.caseID = :caseID 
  AND CaseUserRole.orgObjectLinkID = OrgObjectLink.orgObjectLinkID 
  AND CaseUserRole.typeCode = :typeCode 
  AND CaseUserRole.recordStatus = :recordStatus
