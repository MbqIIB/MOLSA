Database SQL Operation: PlannedSubGoal.searchDetailsByPlannedGroupID

Operation Details
Description: Reads a list of details for all sub-goals related to a specific planned group.
Sterotype: nsmulti

SQL 
SELECT
  PlannedSubGoal.plannedSubGoalID,
  PlannedSubGoal.sensitivityCode,
  PlannedSubGoal.expectedStartDate,
  PlannedSubGoal.expectedEndDate,
  PlannedSubGoal.actualEndDate,
  PlannedSubGoal.actualStartDate,
  SubGoal.name,
  SubGoal.typeCode
INTO
  :plannedSubGoalID,
  :sensitivityCode,
  :expectedStartDate,
  :expectedEndDate,
  :actualEndDate,
  :actualStartDate,
  :name,
  :typeCode 
FROM
  PlannedSubGoal,
  SubGoal 
WHERE PlannedSubGoal.plannedGroupID = :plannedGroupID 
  AND SubGoal.subGoalID = PlannedSubGoal.subGoalID
