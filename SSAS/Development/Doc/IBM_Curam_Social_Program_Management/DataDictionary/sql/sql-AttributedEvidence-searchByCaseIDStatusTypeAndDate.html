Database SQL Operation: AttributedEvidence.searchByCaseIDStatusTypeAndDate

Operation Details
Description: Performs a search across the Attributed Evidence and Evidence Descriptor entities for evidence by type, case, status and date.
Sterotype: nsmulti

SQL 
SELECT
  EvidenceDescriptor.relatedID,
  AttributedEvidence.attributedFromDate,
  AttributedEvidence.attributedToDate
INTO
  :relatedID,
  :attributedFromDate,
  :attributedToDate 
FROM
  AttributedEvidence,
  EvidenceDescriptor 
WHERE AttributedEvidence.caseID = :caseID 
  AND EvidenceDescriptor.evidenceDescriptorID = AttributedEvidence.evidenceDescriptorID 
  AND EvidenceDescriptor.evidenceType = :evidenceType 
  AND EvidenceDescriptor.statusCode = :statusCode 
  AND 
  (
    AttributedEvidence.attributedFromDate <= :calculationDate 
    OR AttributedEvidence.attributedFromDate IS NULL 
  ) 
  AND 
  (
    AttributedEvidence.attributedToDate >= :calculationDate 
    OR AttributedEvidence.attributedToDate IS NULL 
  )
