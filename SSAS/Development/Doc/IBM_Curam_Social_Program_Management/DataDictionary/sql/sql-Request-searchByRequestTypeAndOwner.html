Database SQL Operation: Request.searchByRequestTypeAndOwner

Operation Details
Description: Returns a list of request details based on request type and user name specified.
Sterotype: nsmulti

SQL 
SELECT
  requestID,
  requestType,
  requestCategory,
  requestStatus,
  R.concernRoleID,
  dateCreated,
  dateSubmitted,
  rejectionReason,
  comments,
  R.versionNo
INTO
  :requestID,
  :requestType,
  :requestCategory,
  :requestStatus,
  :concernRoleID,
  :dateCreated,
  :dateSubmitted,
  :rejectionReason,
  :comments,
  :versionNo 
FROM
  Request R INNER JOIN
  
  AdministrationConcernRole 
  ON (R.concernRoleID = AdministrationConcernRole.concernRoleID) INNER JOIN
  
  AdministrationRole 
  ON (AdministrationConcernRole.administrationRoleID = AdministrationRole.administrationRoleID) 
WHERE 
  (
    requestType = :reqType 
  ) 
  AND 
  (
    AdministrationRole.userName = :userName 
  )
