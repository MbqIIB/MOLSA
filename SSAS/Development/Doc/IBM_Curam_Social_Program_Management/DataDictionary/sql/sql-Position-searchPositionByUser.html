Database SQL Operation: Position.searchPositionByUser

Operation Details
Description: Method to search for all active positions associated with a user.
Sterotype: nsmulti

SQL 
SELECT
  Position.positionID,
  Position.name
INTO
  :positionID,
  :positionName 
FROM
  Position,
  PositionHolderLink 
WHERE PositionHolderLink.organisationStructureID = :orgStructureID 
  AND PositionHolderLink.userName = :userName 
  AND Position.positionID = PositionHolderLink.positionID 
  AND PositionHolderLink.recordStatus = :statusCode 
  AND PositionHolderLink.fromDate <= :effectiveDate 
  AND 
  (
    PositionHolderLink.toDate >= :effectiveDate 
    OR PositionHolderLink.toDate IS NULL 
  )
