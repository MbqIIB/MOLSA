Database SQL Operation: ExternalParty.searchByNextPmtDateRangeMOP

Operation Details
Description: Operation to retrieve all External Party records that match the specified method of payment code and where the next payment date is within the specified date range.
Sterotype: nsmulti

SQL 
SELECT
  ExternalParty.concernRoleID,
  ExternalParty.name,
  ExternalParty.primaryAlternateID,
  ExternalParty.verificationInd,
  ExternalParty.paymentFrequency,
  ExternalParty.methodOfPmtCode,
  ExternalParty.nextPaymentDate,
  ExternalParty.currencyType,
  ExternalParty.type,
  ExternalParty.status,
  ExternalParty.comments,
  ExternalParty.versionNo
INTO
  :concernRoleID,
  :name,
  :primaryAlternateID,
  :verificationInd,
  :paymentFrequency,
  :methodOfPmtCode,
  :nextPaymentDate,
  :currencyType,
  :type,
  :status,
  :comments,
  :versionNo 
FROM
  ExternalParty 
WHERE ExternalParty.methodOfPmtCode = :methodOfPmtCode 
  AND ExternalParty.nextPaymentDate >= :nextPmtDateFrom 
  AND ExternalParty.nextPaymentDate <= :nextPmtDateTo
