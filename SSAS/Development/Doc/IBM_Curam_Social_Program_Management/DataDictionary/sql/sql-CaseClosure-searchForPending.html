Database SQL Operation: CaseClosure.searchForPending

Operation Details
Description: Method which reads records whose closure date is before the date supplied and whose associated case has the status supplied
Sterotype: nsmulti

SQL 
SELECT
  CaseClosure.caseClosureID,
  CaseClosure.caseID,
  CaseClosure.notesID,
  CaseClosure.closureDate,
  CaseClosure.reasonCode,
  CaseClosure.recordStatus,
  CaseClosure.versionNo
INTO
  :caseClosureID,
  :caseID,
  :notesID,
  :closureDate,
  :reasonCode,
  :recordStatus,
  :versionNo 
FROM
  CaseClosure,
  CaseHeader 
WHERE CaseClosure.closureDate <= :closureDate 
  AND CaseClosure.caseID = CaseHeader.caseID 
  AND CaseHeader.statusCode = :statusCode
  AND CaseHeader.endDate = CaseClosure.closureDate
