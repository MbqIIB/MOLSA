Database SQL Operation: Provider.searchLiabilitiesByConcernRole

Operation Details
Description: Searches&nbsp;and&nbsp;returns&nbsp;a&nbsp;list of&nbsp;liability financial instruction based on the concernrole and case nominee.
Sterotype: nsmulti

SQL 
SELECT LIABILITYINSTRUCTION.FININSTRUCTIONID,
FINANCIALINSTRUCTION.TYPECODE,
LIABILITYINSTRUCTION.CASENOMINEEID,
LIABILITYINSTRUCTION.AMOUNT,
CONCERNROLE.CONCERNROLEID
INTO
:finInstructionID, :typeCode, :caseNomineeID, :amount, :concernRoleID
FROM LIABILITYINSTRUCTION,CASENOMINEE,CASEPARTICIPANTROLE,
CONCERNROLE,FINANCIALINSTRUCTION
WHERE LIABILITYINSTRUCTION.CASENOMINEEID = CASENOMINEE.CASENOMINEEID
AND CASEPARTICIPANTROLE.CASEPARTICIPANTROLEID = CASENOMINEE.CASEPARTICIPANTROLEID
AND CONCERNROLE.CONCERNROLEID = CASEPARTICIPANTROLE.PARTICIPANTROLEID
AND LIABILITYINSTRUCTION.FININSTRUCTIONID = FINANCIALINSTRUCTION.FININSTRUCTIONID
AND CONCERNROLE.CONCERNROLEID=:concernRoleID
