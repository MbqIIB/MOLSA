Database SQL Operation: Hearing.readDeskHearingSummaryDetails

Operation Details
Description: Reads the summary details for a desk-based hearing.
Sterotype: ns

SQL 
SELECT
  Hearing.versionNo,
  Hearing.typeCode,
  Hearing.scheduledDateTime,
  Hearing.statusCode,
  OrgObjectLink.orgObjectLinkID,
  Hearing.comments,
  Hearing.referenceNumber,
  Hearing.noticeDate
INTO
  :versionNo,
  :typeCode,
  :scheduledDateTime,
  :statusCode,
  :orgObjectLinkID,
  :comments,
  :referenceNumber,
  :noticeDate 
FROM
  Hearing,
  HearingUserRole,
  CaseUserRole,
  OrgObjectLink 
WHERE Hearing.hearingID = :hearingID 
  AND Hearing.hearingID = HearingUserRole.hearingID 
  AND HearingUserRole.caseUserRoleID = CaseUserRole.caseUserRoleID 
  AND CaseUserRole.typeCode = :typeCode 
  AND CaseUserRole.orgObjectLinkID = OrgObjectLink.orgObjectLinkID
