Database View: DW_DIMCASESUPER

View Details
Description: DW_DIMCASESUPER : Case Super View

Description:

This view returns the following attributes:

Example:

Technical Notes:

SQL 
select  max(cs.DWCASESUPERID) as DWCASESUPERID,
        substr(co.caseownerid, 1, 30) as SUPERNAME,
        us.firstname as FirstName,
        us.lastname as LastName,
        substr(co.CASEOWNERTYPEDESC, 1, 30) as SuperType, 
        max(cs.LASTWRITTEN) as LASTWRITTEN
from    dw_casesuper cs,
        dw_caseowner co,
        dw_users us
where   cs.dwcaseownerid = co.dwcaseownerid 
and     co.CASEOWNERID = us.CASEOWNERID
group by co.caseownerid, us.firstname, us.lastname, co.CASEOWNERTYPEDESC
union
select  max(cs.DWCASESUPERID) as DWCASESUPERID,
        substr(org.name, 1, 30) as SUPERNAME,
        null as FirstName,
        null as LastName,
        substr(co.CASEOWNERTYPEDESC, 1, 30) as SuperType, 
        max(cs.LASTWRITTEN) as LASTWRITTEN
from    dw_casesuper cs,
        dw_caseowner co,
        dw_orgunit org
where   cs.dwcaseownerid = co.dwcaseownerid 
and     co.DWORGUNITID = org.DWORGUNITID
group by org.name, co.CASEOWNERTYPEDESC
