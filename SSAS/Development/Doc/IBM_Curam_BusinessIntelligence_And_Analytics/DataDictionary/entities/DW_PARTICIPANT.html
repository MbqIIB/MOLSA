Database Table: DW_PARTICIPANT

Table Description
DW_PARTICIPANT : Case Participant Central Warehouse Table

Description:

A PARTICIPANT associated with a CASE.

A PARTICIPANT can have multiple case participant roles in a single case.

A PARTICIPANT can have many different roles in different CASES.

A CASE may have one or more case participant roles.
