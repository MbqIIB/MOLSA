Database Table: DM_FACTPAYMENTS

Table Description
DM_FACTPAYMENTS : Payments Fact Table

Description:

This table represents instruction line item data and is the lowest level of financial transaction data in the application. Each instruction line item is a single payment (or bill) in respect of an individual product component for a specific. The grain of this accumulating snapshot is one row per instruction line item. Because this is an accumulating snapshot, each unique record in the fact table is revised and updated whenever financial processing steps are completed.
