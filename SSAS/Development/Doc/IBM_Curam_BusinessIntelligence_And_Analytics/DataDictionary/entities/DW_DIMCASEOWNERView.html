Database View: DW_DIMCASEOWNER

View Details
Description: DW_DIMCASEOWNER : Case Owner View

Description:

This view returns the following attributes:

Example:

Technical Notes:

SQL 
select  co.caseownerid as ownername,
        max(co.DWCASEOWNERID) as DWCASEOWNERID,
        us.firstname as FirstName,
        us.lastname as LastName,
        substr(co.CASEOWNERTYPEDESC, 1, 30) as OwnerType, 
        max(co.LASTWRITTEN) as LASTWRITTEN  
from    dw_caseowner co,
        dw_users us
where   co.CASEOWNERID = us.CASEOWNERID
group by co.caseownerid, us.firstname, us.lastname, co.CASEOWNERTYPEDESC
union
select  substr(org.name, 1, 30) as ownername,
        max(co.DWCASEOWNERID) as DWCASEOWNERID,
        null as FirstName,
        null as LastName,
        substr(co.CASEOWNERTYPEDESC, 1, 30) as OwnerType, 
        max(co.LASTWRITTEN) as LASTWRITTEN  
from    dw_caseowner co,
        dw_orgunit org
where   co.DWORGUNITID = org.DWORGUNITID
group by org.name, co.CASEOWNERTYPEDESC
