Database Table: DM_FACTCASEHISTORY

Table Description
DM_FACTCASEHISTORY : Case History Fact Table

Description:

This fact tables stores a history of the status of a case as it moves through the different states, e.g. Open, Submitted, Active, Cancelled, Closed.
