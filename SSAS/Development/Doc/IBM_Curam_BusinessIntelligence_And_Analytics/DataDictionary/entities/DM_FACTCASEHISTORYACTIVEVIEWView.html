Database View: DM_FACTCASEHISTORYACTIVEVIEW

View Details
Description: DM_FACTCASEHISTORYACTIVEVIEW : Active Cases View

Description:

View containing all cases which have a status of Active.

Example:

Technical Notes:

SQL 
select
t1.CASEID  as caseid,
nvl(t1.startdatekey,-1)  as activestartdatekey,
nvl(t1.enddatekey,-1)  as activeenddatekey
from
DM_FACTCASEHISTORY t1,
DM_DIMSTATUS t2
where
t2.STATUSNAME ='Active' and
t1.STATUSKEY = t2.statuskey;
