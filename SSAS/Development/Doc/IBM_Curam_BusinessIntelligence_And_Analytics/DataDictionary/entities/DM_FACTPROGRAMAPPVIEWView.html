Description: DM_FACTPROGRAMAPPVIEW : Fact Program Application View

Description:

This view returns the following attributes:

Example:

Technical Notes:

SQL 
select
   t1.INTAKEPROGRAMAPPLICATIONID,
   t1.DWINTAKETYPEKEY,
   t1.DWINTAKECATEGORYKEY,
   t1.DWRECOMMENDATIONKEY,
   t1.DWRESPONDSWITHINTIMEKEY,
   t1.DWRESPONSEPRIORITYKEY,
   t1.DWDISPOSITIONKEY,
   t1.CASEID,
   t1.DWDENIALREASONKEY,
   t1.INTAKEAPPLICATIONID,
   t1.INTAKESTARTDATE,
   t1.ENTEREDBYUSER,
   t1.DISPOSEDDATETIME,
   t1.NUMHOURSTODISPOSE,
   t1.NUMALLEGATIONS,
   t1.MEETRESPONDSTIMEIND,
   t1.LASTWRITTEN
from
   DW_PROGRAMAPPLICATION t1,
   DW_CASE t2,
   DW_CASEOWNER t3
where
   t1.CASEID = t2.CASEID
   and t2.DWCASEOWNERID = t3.dwcaseownerid;
