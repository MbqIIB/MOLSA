Database Table: DW_PAYMENT

Table Description
DW_PAYMENT : Payments Central Warehouse Table

Description:

This table represents instruction line item data and is the lowest level of financial transaction data in the application. Each payment item is a single payment (or bill) in respect of an individual product component for a specific case.
