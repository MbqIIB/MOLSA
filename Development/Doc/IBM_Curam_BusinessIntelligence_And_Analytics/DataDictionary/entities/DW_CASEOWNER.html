Database Table: DW_CASEOWNER

Table Description
DW_CASEOWNER : Case Owner Central Warehouse Table

Description:

Records the owners of the Case using details from the ORGOBJECTLINK table. Organization Object Link allows the ability to link to an Organization Object which can be any one of USER, ORGANIZATION, POSITION, QUEUE.
