Database Table: DW_PROGRAMAPPLICATION

Table Description
DW_PROGRAMAPPLICATION : Program Application Central Warehouse Table

Description:

An application form may be submitted for one or more programs. This entity represents an application made for a particular program as part of the receipt of an application form (IntakeApplication).
