Database Table: DW_ADDRESS

Table Description
DW_ADDRESS : Address Central Warehouse Table

Description:

An address is a description of a location, as written or printed on mail as directions for delivery. Many concurrent addresses can be stored for a PARTICIPANT but one address is marked as the primary or default address. A type e.g. home, work, etc. can be recorded for each address.
