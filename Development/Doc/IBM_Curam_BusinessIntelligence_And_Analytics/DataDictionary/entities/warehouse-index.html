DW_CONTACTLOG Warehouse core Table DW_CONTACTLOG : Contact Log Central Warehouse Table

Description:

A Contact Log is a record of any interaction or meeting that occurs between the case worker and a client, this tables contains those details. The primary source entity used by this table is LOG.

A LOG is any interaction or meeting that occurs between the WORKER and a CLIENT, which may relate to an INVESTIGATION case. LOGS may occur in many different locations and be conducted face to face or through other media. Each INVESTIGATION may have one or many LOGS which relate to different CLIENTS over the case lifecycle.

Example:

Technical Notes:
DW_CONTACTLOGPURPOSE Warehouse core Table DW_CONTACTLOGPURPOSE : Contact Log Purpose Central Warehouse Table

Description:

A Contact Log Purpose is the reason for an interaction or meeting that occurs between the case worker and a client this tables contains those details. The primary source entity used by this table is PURPOSE.

When a Contact Log is created, it is possible to store multiple purposes for the contact. When searching for a Contact Log, the case worker can choose 1 purpose and any contact which contains this purpose will be returned. To facilitate this, each purpose is recorded separately on this entity.
