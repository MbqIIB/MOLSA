Database Table: S_CASEDEDUCTIONHISTORY

Table Description
S_CASEDEDUCTIONHISTORY : Staging table Case Deduction History

Description:

A deduction history record is created each time a ITEM becomes active, inactive, is processed, skipped or cancelled. A case deduction history record is created for a ITEM that is paying a third party each time the payment is awaiting payment, issued or cancelled.
