DM_FACTUTILITYHISTORY Datamart core Table DM_FACTUTILITYHISTORY : Utility History Fact Table

Description:

This is a fact table for utility providers which records history and changes over time by inserting new rows when a business level event occurs, the source table is UTILITY.

An external organization paid by the human services ORGANIZATION on behalf of a PARTICIPANT.

A utility is a type of PARTICIPANT.

The utility is paid, as instructed by the PARTICIPANT, by deducting money from an active claim entitlement the PARTICIPANT has registered on the system.
