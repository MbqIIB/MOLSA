Domain Definition: PDT_LOCALIZE_TEXT_TYPE

Domain Definition Details
Name: PDT_LOCALIZE_TEXT_TYPE
Component: core
Derived From: SVR_INT32
Options:  
Description: All of the strings that may be localized in a workflow process definition are displayed on the one page in the process definition tool. To allow the process definition tool and the workflow engine to determine which string is being localized, this enumerated type is used. It represents the types of string that may be localized in a workflow process definition (i.e. activity name, activity description etc.).
