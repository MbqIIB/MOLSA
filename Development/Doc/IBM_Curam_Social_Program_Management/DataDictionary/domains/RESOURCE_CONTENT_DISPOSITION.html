Domain Definition: RESOURCE_CONTENT_DISPOSITION

Domain Definition Details
Name: RESOURCE_CONTENT_DISPOSITION
Component: IntelligentEvidenceGathering
Derived From: SVR_STRING<254>
Options:  
Description: The content disposition for a resource. This allows information such as the proposed file name, the creation and modification time stamps, and the content size to be specified. This should conform to IETF RFC 2183, though the header name should be omitted and the disposition-type value (i.e., "inline" or "attachment") will be ignored and applied as appropriate when the resource is used.
