Domain Definition: CW_DOMAIN_NAME

Domain Definition Details
Name: CW_DOMAIN_NAME
Component: CitizenWorkspaceAdmin
Derived From: CODETABLE_CODE
Options: CT: AuthFactDomains
Description: Domain for domain names. This determines how an authentication factor is rendered by the portal. For example a value of CW_PASSWORD causes the factor to be rendered as a password field whereas a value of CW_DATE causes it to be rendered as a date complete with a calendar picker.
