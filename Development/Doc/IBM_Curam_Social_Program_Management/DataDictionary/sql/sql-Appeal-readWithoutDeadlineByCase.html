Database SQL Operation: Appeal.readWithoutDeadlineByCase

Operation Details
Description: Reads the judicial appeal details by caseID without a deadlineTaskID, as a deadline is not required for a judicial review
Sterotype: ns

SQL 
SELECT
  Appeal.appealID,
  Appeal.caseID,
  Appeal.difficultyCode,
  Appeal.appealTypeCode,
  Appeal.deadlineDate,
  Appeal.comments,
  Appeal.versionNo,
  CaseHeader.registrationDate,
  CaseHeader.caseReference,
  CaseHeader.receivedDate,
  CaseHeader.statusCode,
  CaseHeader.versionNo
INTO
  :appealID,
  :caseID,
  :difficultyCode,
  :appealTypeCode,
  :deadlineDate,
  :comments,
  :appealVersionNo,
  :registrationDate,
  :caseReference,
  :receivedDate,
  :statusCode,
  :caseVersionNo 
FROM
  Appeal,
  CaseHeader 
WHERE Appeal.caseID = CaseHeader.caseID 
  AND Appeal.caseID = :caseID
