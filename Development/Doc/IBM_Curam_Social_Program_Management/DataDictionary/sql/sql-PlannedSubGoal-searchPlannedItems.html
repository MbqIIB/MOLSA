Database SQL Operation: PlannedSubGoal.searchPlannedItems

Operation Details
Description: Returns plan item name, plan item type, expected start date, expected end date, outcome name, actual start date actual end date and maximum period indicator of all existing planned items of specified planned sub-goal.
Sterotype: nsmulti

SQL 
SELECT
  PlannedItem.name,
  PlannedItem.expectedStartDate,
  PlannedItem.expectedEndDate,
  PlannedItem.status,
  PlannedItem.outcomeAchieved,
  PlannedItem.actualStartDate,
  PlannedItem.actualEndDate,
  PlannedItem.datePeriodMaxInd,
  PlannedItem.plannedItemID
INTO
  :planItemName,
  :expectedStartDate,
  :expectedEndDate,
  :status,
  :outcomeName,
  :actualStartDate,
  :actualEndDate,
  :datePeriodMaxInd,
  :plannedItemID 
FROM
  PlannedItem,
  PlannedSubGoal 
WHERE PlannedSubGoal.plannedSubGoalID = PlannedItem.plannedSubGoalID 
  AND PlannedSubGoal.plannedSubGoalID = :plannedSubGoalID 
ORDER BY 
  PlannedItem.createdOn
