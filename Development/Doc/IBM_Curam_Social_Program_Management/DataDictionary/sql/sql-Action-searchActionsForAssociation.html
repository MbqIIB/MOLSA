Database SQL Operation: Action.searchActionsForAssociation

Operation Details
Description: Method to read Action details list records for specified parentLinkID and not linked with specified situationID.
Sterotype: nsmulti

SQL 
SELECT
  Action.actionID,
  action
INTO
  :actionID,
  :action 
FROM
  Action 
WHERE Action.actionID NOT IN 
  ( 
  SELECT
    actionID 
  FROM
    ActionSituationLink 
  WHERE ActionSituationLink.situationID = :situationID 
    AND recordStatus = :recordStatus 
  ) 
  AND Action.actionID IN 
  ( 
  SELECT
    ActionParentLink.actionID 
  FROM
    ActionParentLink 
  WHERE parentLinkID = :parentLinkID 
    AND parentType = :parentType 
  ) 
 AND Action.recordStatus = :recordStatus
