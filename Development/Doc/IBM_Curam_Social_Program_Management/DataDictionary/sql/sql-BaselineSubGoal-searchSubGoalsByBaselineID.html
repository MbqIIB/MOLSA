Database SQL Operation: BaselineSubGoal.searchSubGoalsByBaselineID

Operation Details
Description: Returns id, name and type from baseline sub-goal for a specified baseline identifier.
Sterotype: nsmulti

SQL 
SELECT
  BaselineSubGoal.baselineSubGoalID,
  BaselineSubGoal.name,
  BaselineSubGoal.typeCode,
  BaselineSubGoal.plannedSubGoalID
INTO
  :baselineSubGoalID,
  :name,
  :typeCode,
  :plannedSubGoalID 
FROM
  BaselineSubGoal 
WHERE BaselineSubGoal.baseLineID = :baseLineID
