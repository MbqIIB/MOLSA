Database SQL Operation: PRLISALILink.searchRostersForSALIByPRLIStatuses

Operation Details
Description: Searches the list of rosters for a service authorization line item, on which the related roster line item by given two statuses sorted by the roster start date.
Sterotype: nsmulti

SQL 
SELECT roster.rosterID INTO :rosterID 

FROM PRLISALILink , ProviderRosterLineItem, RosterLineItem, Roster

WHERE
  
PRLISALILink.serviceAuthorizationLIID = :saliID AND
  
providerRosterLineItem.providerRosterLineItemID = PRLISALILink.providerRosterLineItemID AND
  
rosterLineItem.rosterLineItemID = providerRosterLineItem.rosterLineItemID  AND
  
roster.rosterID = rosterLineItem.rosterID AND
  
(providerRosterLineItem.status = :prliStatus1 OR providerRosterLineItem.status = :prliStatus2)
  
ORDER BY roster.fromDate
