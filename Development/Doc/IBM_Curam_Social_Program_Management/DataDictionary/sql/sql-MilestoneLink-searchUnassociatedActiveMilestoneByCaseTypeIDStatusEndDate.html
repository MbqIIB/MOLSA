Database SQL Operation: MilestoneLink.searchUnassociatedActiveMilestoneByCaseTypeIDStatusEndDate

Operation Details
Description: @Deprecated since curam 6.0, replaced by {@Link MiestoneLink#searchUnassociatedActiveMilestonesByCaseTypeIDStatusEndDate() } See release note: CR00236426 This method lists all the active milestone not already associated with a particular type of case.
Sterotype: nsmulti

SQL 
SELECT
  milestoneConfigurationID,
  name,
  type,
  startDate,
  endDate,
  creationDate
INTO
  :milestoneConfigurationID,
  :name,
  :type,
  :startDate,
  :endDate,
  :creationDate 
FROM
  MilestoneConfiguration 
WHERE milestoneConfigurationID NOT IN 
  ( 
  SELECT
    milestoneConfigurationID 
  FROM
    MilestoneLink 
  WHERE caseTypeID = :caseTypeID 
  ) 
  AND recordStatus = :recordStatus 
  AND 
  (
    endDate >= :endDate 
    OR endDate IS NULL 
  )
