Database SQL Operation: IntegratedCaseMemberNote.readNote1

Operation Details
Description: Reads Integrated Case Member Note details.
Sterotype: ns

SQL 
SELECT
  IntegratedCaseMemberNote.integratedCaseMemberNoteID,
  IntegratedCaseMemberNote.participantID,
  IntegratedCaseMemberNote.recordStatus,
  IntegratedCaseMemberNote.caseID,
  Note.sensitivityCode,
  Note.notesText,
  Note.subjectText,
  Note.creationDateTime,
  Note.priorityCode,
  Users.fullName,
  Note.userName,
  IntegratedCaseMemberNote.noteID,
  IntegratedCaseMemberNote.versionNo
INTO
  :integratedCaseMemberNoteID,
  :participantID,
  :recordStatus,
  :caseID,
  :sensitivityCode,
  :notesText,
  :subjectText,
  :creationDateTime,
  :priorityCode,
  :fullName,
  :userName,
  :noteID,
  :versionNo 
FROM
  Note,
  IntegratedCaseMemberNote,
  Users 
WHERE IntegratedCaseMemberNote.integratedCaseMemberNoteID = :integratedCaseMemberNoteID 
  AND Note.noteID = IntegratedCaseMemberNote.noteID 
  AND Note.userName = Users.userName
