Database SQL Operation: AdministrationCaseRole.searchByCaseRoleAndStatus

Operation Details
Description: Method that reads all administration case roles for the specified case identifier and status values.
Sterotype: nsmulti

SQL 
SELECT
  AdministrationCaseRole.caseID,
  AdministrationCaseRole.userName,
  AdministrationCaseRole.startDate,
  AdministrationCaseRole.endDate,
  AdministrationCaseRole.typeCode,
  AdministrationCaseRole.statusCode,
  AdministrationCaseRole.reasonCode,
  AdministrationCaseRole.comments,
  AdministrationCaseRole.administrationCaseRoleID,
  AdministrationCaseRole.versionNo
INTO
  :caseID,
  :userName,
  :startDate,
  :endDate,
  :typeCode,
  :statusCode,
  :reasonCode,
  :comments,
  :administrationCaseRoleID,
  :versionNo 
 FROM
  AdministrationCaseRole 
WHERE caseID = :caseID 
  AND 
  (
    statusCode = :status1 
    OR statusCode = :status2 
  )
