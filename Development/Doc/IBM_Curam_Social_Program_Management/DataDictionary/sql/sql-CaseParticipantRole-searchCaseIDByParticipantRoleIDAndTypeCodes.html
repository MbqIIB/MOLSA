Database SQL Operation: CaseParticipantRole.searchCaseIDByParticipantRoleIDAndTypeCodes

Operation Details
Description: This method retrieves all case IDs from the CaseParticipantRole table that match the given participant role ID and either one of two type codes.
Sterotype: nsmulti

SQL 
SELECT
  caseID
INTO
  :caseID 
FROM
  CaseParticipantRole 
WHERE 
  (
    CaseParticipantRole.typeCode = :typeCode1 
    OR CaseParticipantRole.typeCode = :typeCode2 
  ) 
  AND CaseParticipantRole.participantRoleID = :participantRoleID
