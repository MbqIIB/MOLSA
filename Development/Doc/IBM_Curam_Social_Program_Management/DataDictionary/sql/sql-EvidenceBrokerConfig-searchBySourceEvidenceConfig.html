Database SQL Operation: EvidenceBrokerConfig.searchBySourceEvidenceConfig

Operation Details
Description: Returns all evidence broker configuration records for a given source type, source ID, evidence type, and record status.
Sterotype: nsmulti

SQL 
SELECT
  evidenceBrokerConfigID,
  sourceType,
  sourceID,
  sourceSystemID,
  targetType,
  targetID,
  sourceEvidenceType,
  targetEvidenceType,
  autoAcceptInd,
  webServicesInd,
  sharedType,
shareVerificationsOption,
  externalSystem,
  recordStatus,
  versionNo
INTO
  :evidenceBrokerConfigID,
  :sourceType,
  :sourceID,
  :sourceSystemID,
  :targetType,
  :targetID,
  :sourceEvidenceType,
  :targetEvidenceType,
  :autoAcceptInd,
  :webServicesInd,
  :sharedType,
  :shareVerificationsOption,
  :externalSystem,
  :recordStatus,
  :versionNo 
FROM
  EvidenceBrokerConfig 
WHERE 
(
      sourceID = :sourceID 
      OR 
      sourceID IS NULL
)
AND 
 (
    sourceType = :sourceType 
    OR 
    sourceType IS NULL 
  )
  AND 
  (
    sourceEvidenceType = :sourceEvidenceType 
    OR 
    sourceEvidenceType IS NULL 
  ) 
  AND
 (
      sourceSystemID = :sourceSystemID 
      OR 
      sourceSystemID IS NULL
  )
  AND recordStatus = :recordStatus 
  ORDER BY targetID,
  targetType,
  targetEvidenceType
