Database SQL Operation: ProgramAppStatusTransition.searchByProgAppIDandProgAppStatus

Operation Details
Description: Retrieves a list of record by the programApplicationID and the programApplication status.
Sterotype: nsmulti

SQL 
SELECT
  ProgramAppStatusTransition.programAppStatusTransitionID,
  ProgramAppStatusTransition.comments,
  ProgramAppStatusTransition.programApplicationStatusID,
  ProgramAppStatusTransition.reason,
   ProgramAppStatusTransition.versionNo
INTO
  :programAppStatusTransitionID,
  :comments,
  :programApplicationStatusID,
  :reason,
  :versionNo
FROM
 ProgramAppStatusTransition
INNER JOIN ProgramApplicationStatus on ProgramApplicationStatus.ProgramApplicationStatusID =  ProgramAppStatusTransition.ProgramApplicationStatusID 
INNER JOIN ProgramApplication on ProgramApplication.ProgramApplicationID = ProgramApplicationStatus.ProgramApplicationID
WHERE 
ProgramApplication.programApplicationID = ProgramApplicationStatus.programApplicationID
and ProgramApplicationStatus.programApplicationID = :programApplicationID
and ProgramApplicationStatus.status = :programApplicationStatus
ORDER BY ProgramApplicationStatus.dateTime;
