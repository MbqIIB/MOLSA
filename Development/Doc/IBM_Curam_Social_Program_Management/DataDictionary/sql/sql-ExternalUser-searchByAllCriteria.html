Database SQL Operation: ExternalUser.searchByAllCriteria

Operation Details
Description: Searches for all external users based on the partial match criteria provided.
Sterotype: nsmulti

SQL 
SELECT
  userName,
  firstname,
  surname,
  fullName,
  statusCode,
  roleName,
  defaultLocale
INTO
  :username,
  :firstname,
  :surname,
  :fullName,
  :statusCode,
  :roleName,
  :defaultLocale 
FROM
  ExternalUser 
WHERE 
  (
    UPPER(firstname) like :firstname 
    OR :firstname IS NULL 
  ) 
  AND 
  (
    UPPER(surname) like :surname 
    OR :surname IS NULL 
  ) 
  AND 
  (
    statusCode = :statusCode 
    OR :statusCode IS NULL 
  ) 
  AND 
  (
    type = :type 
    OR :type IS NULL 
  ) 
  AND 
  (
    UPPER (userName) like :userName 
    OR :userName IS NULL 
  ) 
  AND 
  (
    roleName = :roleName 
    OR :roleName IS NULL 
  )
