Database SQL Operation: DiscussionPost.searchPostsByDiscussion

Operation Details
Description:  
Sterotype: nsmulti

SQL 
SELECT
  post.postID,
  post.discussionID,
  post.postComment,
  post.postedByDateTime,
  post.postedByUser,
  post.lastEditedDateTime,
  post.lastEditedByUser,
  post.parentPostID,
  post.versionNo
INTO
  :postID,
  :discussionID,
  :postComment,
  :postedByDateTime,
  :postedByUser,
  :lastEditedDateTime,
  :lastEditedByUser,
  :parentPostID,
  :versionNo 
FROM
  DiscussionPost post 
WHERE post.discussionID = :discussionID 
  AND post.postID ^ = 
  ( 
  SELECT
    discussionHeaderPostID 
  FROM
    Discussion dis 
  WHERE dis.discussionID = post.discussionID 
  ) 
ORDER BY post.postedByDateTime
