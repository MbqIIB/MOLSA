Database SQL Operation: FundLink.searchByRelatedIDAndPriorityAndDate

Operation Details
Description: This method searches and returns all the record for the given related id, start and end dates in the order of priority.
Sterotype: nsmulti

SQL 
SELECT
  FundLink.fundLinkID,
  FundLink.programFundID,
  FundLink.relatedType,
  FundLink.relatedID,
  FundLink.priority,
  FundLink.startDate,
  FundLink.endDate,
  FundLink.recordStatus,
  FundLink.comments
INTO
  :fundLinkID,
  :programFundID,
  :relatedType,
  :relatedID,
  :priority,
  :startDate,
  :endDate,
  :recordStatus,
  :comments 
FROM
  FundLink 
WHERE FundLink.relatedID = :relatedID 
  AND FundLink.startDate <= :startDate 
  AND 
  (
    FundLink.endDate >= :startDate 
    OR FundLink.endDate IS NULL 
  ) 
  AND FundLink.recordStatus = :recordStatus 
ORDER BY FundLink.priority
