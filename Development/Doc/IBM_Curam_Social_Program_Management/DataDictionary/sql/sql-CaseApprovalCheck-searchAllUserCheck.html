Database SQL Operation: CaseApprovalCheck.searchAllUserCheck

Operation Details
Description: Retrieves details of all case approval checks for the specified User.
Sterotype: nsmulti

SQL 
SELECT
  CaseApprovalCheck.caseApprovalCheckID,
  CaseApprovalCheck.typeCode,
  CaseApprovalCheck.percentage,
  CaseApprovalCheck.estimatedCost,
  CaseApprovalCheck.productID,
  CaseApprovalCheck.statusCode,
  CaseApprovalCheck.comments,
  Product.name,
  CaseApprovalCheck.versionNo
INTO
  :caseApprovalCheckID,
  :typeCode,
  :percentage,
  :estimatedCost,
  :productID,
  :statusCode,
  :comments,
  :productName,
  :versionNo
FROM
  CaseApprovalCheck 
  
  LEFT OUTER JOIN
  
  Product 
  ON (Product.productID = CaseApprovalCheck.productID) 
WHERE CaseApprovalCheck.userName = :userName
