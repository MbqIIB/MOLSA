Database SQL Operation: CaseDecision.searchDecisionObjectiveDetailsByInstanceID

Operation Details
Description:  
Sterotype: nsmulti

SQL 
SELECT
  CaseDecision.caseDecisionID,
  CaseDecision.tagValue,
  CaseDecisionObjective.objectiveID,
  CaseDecisionObjective.concernRoleID,
  ConcernRole.concernRoleName
INTO
  :caseDecisionID,
  :tagValue,
  :objectiveID,
  :concernRoleID,
  :concernRoleName 
FROM
  CaseDecisionSet,
  CaseDecision,
  CaseDecisionObjective,
  ConcernRole 
WHERE CaseDecisionSet.instanceID = :instanceID 
  AND CaseDecision.caseDecisionID = CaseDecisionSet.caseDecisionID 
  AND CaseDecisionObjective.caseDecisionID = CaseDecision.caseDecisionID 
  AND CaseDecisionObjective.concernRoleID = ConcernRole.concernRoleID 
ORDER BY CaseDecision.caseDecisionID,
  ConcernRole.concernRoleID
