Database SQL Operation: PAApprovalRequest.readRejectionDetailsForModify

Operation Details
Description: Reads planned item rejection details for modify.
Sterotype: ns

SQL 
SELECT
  ApprovalRequest.approvalRequestID,
  ApprovalRequest.versionNo,
  ApprovalRequest.rejectionReason,
  ApprovalRequest.rejectionComments,
  PlannedGoal.caseID
INTO
  :approvalRequestID,
  :versionNo,
  :rejectionReason,
  :rejectionComments,
  :caseID 
FROM
  PAApprovalRequest,
  ApprovalRequest,
  PlannedItem,
  PlannedSubGoal,
  PlannedGoal 
WHERE PAApprovalRequest.plannedItemApprovalRequestID = :plannedItemApprovalRequestID 
  AND ApprovalRequest.approvalRequestID = PAApprovalRequest.approvalRequestID 
  AND PlannedItem.plannedItemID = PAApprovalRequest.plannedItemID 
  AND PlannedSubGoal.plannedSubGoalID = PlannedItem.plannedSubGoalID 
  AND PlannedGoal.plannedGoalID = PlannedSubGoal.plannedGoalID
