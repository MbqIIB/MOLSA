Database SQL Operation: Location.searchActiveLocationsByType

Operation Details
Description: Search locations that are active and operating, excluding the location (and its sub-location) from which additional access is to be assigned. This list also excludes locations which have already been assigned rights.
Sterotype: nsmulti

SQL 
SELECT
  Location.locationID,
  Location.name,
  Location.locationStatus
INTO
  :locationID,
  :locationName,
  :locationStatus 
FROM
  Location 
WHERE Location.statusCode = :statusCode 
  AND Location.locationStructureID = :locationStructureID 
  AND Location.locationStatus = :locationStatus 
  AND Location.locationType = :locationType 
  AND Location.locationID <> :locationID 
  AND Location.locationID NOT IN 
  ( 
  SELECT
    LocationAccess.additionalLocationID 
  FROM
    LocationAccess 
  WHERE LocationAccess.locationID = :locationID 
    AND LocationAccess.maintainInd = '1' 
  )
