Database SQL Operation: CaseEvent.searchEventsByTypeDate

Operation Details
Description: Returns event by type and date. Used in CaseEventsReport. To return all events set eventType and eventTypeAll to the same value. currently a string constant of "ALL" is used in server code.
Sterotype: nsmulti

SQL 
SELECT
  caseEventID,
  caseID,
  relatedID,
  eventTypeCode,
  startDate,
  endDate,
  statusCode,
  recordStatus,
  versionNo
INTO
  :caseEventID,
  :caseID,
  :relatedID,
  :eventTypeCode,
  :startDate,
  :endDate,
  :statusCode,
  :recordStatus,
  :versionNo 
FROM
  CaseEvent 
WHERE startDate >= :fromDate 
  AND startDate <= :toDate 
  AND 
  (
    eventTypeCode = :eventType 
    OR eventTypeCode = :eventTypeAll 
  )
