Database SQL Operation: HearingPhoneContact.searchPhoneContacts

Operation Details
Description: Returns a list of contacts for the specified hearing.
Sterotype: nsmulti

SQL 
SELECT
  HearingPhoneContact.caseParticipantRoleID,
  ConcernRole.concernRoleName,
  PhoneNumber.phoneCountryCode,
  PhoneNumber.phoneAreaCode,
  PhoneNumber.phoneNumber,
  PhoneNumber.phoneExtension,
  HearingPhoneContact.hearingPhoneContactID,
  HearingPhoneContact.recordStatus,
  HearingPhoneContact.concernRolePhoneNumberID,
  HearingPhoneContact.hearingID
INTO
  :caseParticipantRoleID,
  :concernRoleName,
  :phoneCountryCode,
  :phoneAreaCode,
  :phoneNumber,
  :phoneExtension,
  :hearingPhoneContactID,
  :recordStatus,
  :concernRolePhoneNumberID,
  :hearingID 
FROM
  HearingPhoneContact,
  ConcernRolePhoneNumber,
  ConcernRole,
  PhoneNumber 
WHERE HearingPhoneContact.hearingID = :hearingID 
  AND ConcernRolePhoneNumber.concernRolePhoneNumberID = HearingPhoneContact.concernRolePhoneNumberID 
  AND PhoneNumber.phoneNumberID = ConcernRolePhoneNumber.phoneNumberID 
  AND PhoneNumber.statusCode = 'RST1' 
  AND ConcernRole.concernRoleID = ConcernRolePhoneNumber.concernRoleID
