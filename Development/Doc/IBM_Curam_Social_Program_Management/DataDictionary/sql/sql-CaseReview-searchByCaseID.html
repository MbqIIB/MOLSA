Database SQL Operation: CaseReview.searchByCaseID

Operation Details
Description: Method that retrieves all case review records based on caseID and case review status
Sterotype: nsmulti

SQL 
SELECT
  caseReviewID,
  caseID,
  reviewerID,
  scheduledStartDate,
  typeCode,
  startDate,
  endDate,
  reasonCode,
  outcomeCode,
  statusCode,
  expectCompleteDate,
  administrationCaseRoleID,
  versionNo
INTO
  :caseReviewID,
  :caseID,
  :reviewerID,
  :scheduledStartDate,
  :typeCode,
  :startDate,
  :endDate,
  :reasonCode,
  :outcomeCode,
  :statusCode,
  :expectCompleteDate,
  :administrationCaseRoleID,
  :versionNo 
FROM
  CaseReview 
WHERE caseID = :caseID 
  AND 
  (
    statusCode = :reviewStatus1 
    OR statusCode = :reviewStatus2 
  )
