Database Table: LocationStructure

Table Description
Location Structure: The top level of the location hierarchy.

An location structure contains locations. Locations can be assigned to users, positions organization units. An location structure represents the configuration of the locations for both workflow and reporting purposes.

Example

Midway Social Services is a Human Services organization which has implemented Curam. An location structure is created that contains the location hierarchy. Within the structure five locations are created each of which contains a number of locations.
