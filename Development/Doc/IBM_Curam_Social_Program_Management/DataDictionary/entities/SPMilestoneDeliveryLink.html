Database Table: SPMilestoneDeliveryLink

Table Description
SPMilestoneDeliveryLink: A milestone is a significant event or task which will be completed during the life cycle of a milestone configuration type. A review may take place two weeks from the start of a milestone type and it may need to be send to a plan participant at certain points of the milestone types lifecycle. Milestones can be used to facilitate these events.

This entity is used to store data relating to the plan groups and sub goals to which milestones have been added within a service plan.
