Database Table: Training

Table Description
Training: Configuration details for a training program that an organization can record participation against.

Details of a training course that can be provided to potential provider on behalf of the organization

Example

For example, the organization creates a training course called Communicating Effectively in the Workplace, in order to be able to track which Providers have completed that training.
