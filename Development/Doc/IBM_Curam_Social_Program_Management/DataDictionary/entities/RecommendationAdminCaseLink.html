Database Table: RecommendationAdminCaseLink

Table Description
Recommendation Administration Case Link : Links a recommendation to a particular case of a particular case type. When a case of a particular type is linked to a recommendation, it enables recommendations to be produced for the linked case, for example an outcome plan.

This allows for civilities to be recommended on a case without running an assessment i.e. they are recommended on case creation.

Example

A Self Sufficiency outcome plan is created for James Smith, Linda Smith and their child Robert to identify and address their issues. JOb Search is recommended by the system. This entity allows for the Job Search activity to be recommended on the Self Sufficiency outcome plan.
