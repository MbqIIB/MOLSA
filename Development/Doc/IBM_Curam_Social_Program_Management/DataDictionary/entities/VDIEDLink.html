Database Table: VDIEDLink

Table Description
VDIEDLink: Links evidence to a verifiable data item if evidence is recorded for the verifiable data item.

When evidence is inserted for a piece of data that has been defined as requiring verification, a record is added to this entity.

Example

This record is used to link a piece of evidence that requires verification with the requirement to verify this piece of data.
