Database Table: RosterLineItem

Table Description
RosterLineItem: Information about actual units delivered to a client for a service on a roster.

A roster line item is specific to a client who receives service on a Attendance Roster. A Roster Line Item records units for roster period for a concern Role.

Example

An Attendance Roster is created for Child Care service. A child was scheduled to receive 64 units of Child Care for the period 01/05/2008-08/05/2008 but he/she received 62 units. The actual units are recorded as 62 for this client.
