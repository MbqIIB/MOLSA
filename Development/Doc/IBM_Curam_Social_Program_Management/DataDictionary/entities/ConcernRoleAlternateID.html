Database Table: ConcernRoleAlternateID

Table Description
Concern Role Alternate ID: A unique identifier for a participant.

Most organizations use a unique identifier for a participant in their dealings with that participant e.g. on correspondence.

A primary alternate ID is set for each participant. However many alternate ID records can be maintained for each participant.

Example

Examples of primary alternate IDs are:

  • NINO (National Insurance Number) in the UK; and
  • SSN (Social Security Number) in the US.

Examples of other alternate IDs are:

  • a passport number; and
  • a medical card number.

Technical Notes

The concern role ID attribute is the internal identifier used in Curam to identify a concern role. This ID is internal to Curam and is not displayed in the application.
