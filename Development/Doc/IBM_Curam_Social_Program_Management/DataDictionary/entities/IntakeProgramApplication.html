Database Table: IntakeProgramApplication

Table Description
IntakeProgramApplication : An application made by a citizen for a program. An application form may be submitted for one or more programs. This entity represents an application made for a particular program as part of the receipt of an application form (IntakeApplication).

Example

James Smith submits a Social Benefits application form for Rent Allowance and Food Allowance programs online. When he submits the application form, an IntakeProgramApplication record is created for the Rent Allowance and Food Allowance programs that he is applying for.
