Database Table: EmailAddress

Table Description
Email Address: An address used to direct the sending and receiving messages electronically over a computer network.

Example

Two email addresses are stored for John Smith:

  • his personal email address; and
  • his work email address.

Technical Notes

Many types of Curam entities can have a related email address; logically these entities have an 'email address' attribute which is a 'complex data type'. Physically however each of these entities has an email address attribute which relates to a row on the email address table.
