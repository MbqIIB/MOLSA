Database Table: ProviderRosterLineItem

Table Description
Provider Roster Line Item: Roster Line Item information for a Provider.

Information about actual units delivered to a client for a service on a roster.

Example

For example, a child was scheduled to receive 64 units of Child Care for the period 01/05/2008-08/05/2008 but he/she received 62 units.
