Database Table: CREOLEMigrationControl

Table Description
Rule Set Migration Control : A single control row to ensure that no more than one process can concurrently add, modify or remove Rule Sets.

The single control row is locked for update during any rule set migration processing.

Technical Notes

There is no primary key defined because the entity only ever holds a single row.

The single row is populated by a DMX file supplied by Cúram and must not be populated from any other source.
