Database Table: Activity

Table Description
Activity: An event scheduled for a date and time and pertaining to a specific subject.

An activity can be scheduled for one or more users and can be associated with a participant and/or a case. When scheduling an activity, the user can select a location where the activity will take place.

Example

User Ronan Rooney creates an activity to schedule a team meeting to review an open case for participant James Smith. The activity is scheduled for Jan 10th 2004 at 10:00am and will last one hour.
