Database Table: CaseTransactionLog

Table Description
CaseTransactionLog: Case Transaction log stores all the significant transactions of a case. For example: Communications sent, changes in evidence, Payments sent and received. CaseTransactionLog. .

CaseTransactionLog entity is used for generating the log and displaying the same on some particular actions performed by the case worker on different type of cases. The action includes usual operations like Adding/Editing/Canceling Evidences,Certification, Case participants Etc. CaseTransactionLog entity would be able to provide the information about the type of action performed, the user involved and date/time of the action performed.
