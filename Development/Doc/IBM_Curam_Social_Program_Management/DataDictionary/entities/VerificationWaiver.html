Database Table: VerificationWaiver

Table Description
This Entity is used to override/bypass a mandatory verification requirement of a verifiable item. On activation of an evidence with a mandatory verification required for an item have a corresponding record, irrespective of the period of waiver, will get activated. The case rules should take care of what happens to the evidence when outside the waiver period. this entity can be populated manually from the evidence verificatgion screens or through code by implementing curam.verification.sl.infrastructure.impl.EvidenceVerificationWaiver interface and binding it using guice.
