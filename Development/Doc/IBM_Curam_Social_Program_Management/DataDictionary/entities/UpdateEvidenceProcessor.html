Database Table: UpdateEvidenceProcessor

Table Description
Update Evidence Processor : Allows custom processing on Holding Evidence XML data. Update Evidence Processor instances will be invoked as part of Holding Evidence creation.

Example

A citizen has logged into the Citizen Self Service portal and submitted a life event.
