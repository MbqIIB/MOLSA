Database Table: HearingTranscriptionRequest

Table Description
Hearing Transcription Request: A request for a recorded copy of a hearing.

After a hearing is scheduled for an appeal case user can request for a copy of transcription of a hearing @curam.entity appeal.sl.entity.Hearing hearing}. The details of the requester and transcriber are recorded in the system.

Example

A request is received from an appellant Linda Smith for a copy of a transcription of a hearing. The details for the requester Linda Smith and the transcriber are recorded in the system. The transcriber may already be registered as a service supplier in the system.
