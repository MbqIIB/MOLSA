Database Table: ConcernRoleCommunication

Table Description
Concern Role Communication: A communication issued to or received from a participant.

One or more communications can be recorded for a participant.

For each communication the method e.g. phone, email, hard, etc., the type e.g. application form, invoice, etc. is recorded.

Example

An informational leaflet is sent to John Smith advising him of changes to the payment rates for a program called 'sickness benefit' rated for the following year.

The communication direction is 'outgoing' and the method is 'hard copy'.
