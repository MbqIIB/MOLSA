Database Table: OverUnderPaymentBreakdown

Table Description
Over Under Payment Breakdown : A header for the results of a case reassessment per nominee per component, i.e. any over underpayment for a single nominee for a child benefit component.

There may be one or more of these records for each NomineeOverUnderPayment.
