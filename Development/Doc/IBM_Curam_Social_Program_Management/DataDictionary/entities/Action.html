Database Table: Action

Table Description
Action : An activity which is included in an action plan which is associated with addressing an issue identified within the investigation.

An action may be directly associated with an situation within an investigation or may just be part of an overall action plan in an investigation.

Example

Examples of action activities are accessing bank records and interviewing colleagues.
