Database Table: VerificationItemUtilization

Table Description
Verification Item Utilization: A means of relating verification items to verifiable data items.

This entity allows an organization to specify that a verification item can be used to verify a piece of data.

Example

A date of birth can be verified using a number of verification items:

  • Driver's License
  • Passport
  • Birth Certificate
