Database Table: Placement

Table Description
Placement: Client placement information for a place.

Client placement information for a place.

Example

For example, John Smith is a 15 year old boy who is to be placed in Lake County Detention Center for drug abuse starting from 15/02/2007 for six months. Cell 4 in Building A is allocated to John starting from 15/02/2007 till 14/08/2007. This placement information is recorded against the place Cell 4.
