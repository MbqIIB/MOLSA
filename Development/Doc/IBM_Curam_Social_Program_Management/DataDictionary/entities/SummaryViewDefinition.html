Database Table: SummaryViewDefinition

Table Description
Summary View Definition: Stores configurations for different types of summary views for each product

A Product having Rule Set can choose to define Summary View Configuration to view rules summary after each rules execution. A summary view configuration stores Rules Data Object names and attributes mapping to help display summary information.
