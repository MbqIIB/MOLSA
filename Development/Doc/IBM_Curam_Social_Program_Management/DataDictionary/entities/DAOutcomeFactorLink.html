Database Table: DAOutcomeFactorLink

Table Description
DA Outcome Factor Link : Links a Decision Assist Outcome to a factor or a factor classification.

When Cúram Decision Assist is used to derive assessment results the outcomes defined in CDA must be mapped to the factor or factor classifications. This allows CDA to be used to derive the results but allows the classifications defined for a factor to be displayed in the results.
