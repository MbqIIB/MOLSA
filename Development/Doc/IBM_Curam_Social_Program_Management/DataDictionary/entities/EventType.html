Database Table: EventType

Table Description
Event Type: The types of events that can be raised from the Curam application.

This table stores the business event types that can be raised from the Curam application. The combination of Event Class and Event Type will denote the specific event that can be raised.

Example

You require a workflow Process Definition to be enacted whenever payments increase on a case.

You use the Admin application to create an Event Class of 'CASE' to represent events associated with cases. You then add an Event Type of 'PAYMENT_INCREASE' to represent the event associated with the payments increase on a case.

You use the Admin application to read this table and provide a list of available events which can be raised and configure your workflow to be started whenever the 'payment increase' event occurs.
