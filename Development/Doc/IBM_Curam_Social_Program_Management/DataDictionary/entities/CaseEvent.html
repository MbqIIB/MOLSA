Database Table: CaseEvent

Table Description
Case Event: An occurrence during the lifecycle of a case.

Case events can be configured to be displayed on the case calendar.

Example

A case review is a type of case event.

User Ronan Rooney creates a case review and assigns it his supervisor. The case event will be displayed on the case calendar.

Technical Notes

Case event can be thought of as an 'abstract base class' for a number of 'concrete' event entities.
