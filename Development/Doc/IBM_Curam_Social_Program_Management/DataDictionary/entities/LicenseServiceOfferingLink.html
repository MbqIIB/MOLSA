Database Table: LicenseServiceOfferingLink

Table Description
License Service Offering Link: Information relating to the service offerings covered by a license.

A Service Offering can be related to one or more License and a License can have one or more Service Offerings.

Example

The 'State Foster Care License' should have the appropriate service 'Foster Care' which is to be delivered from the provider.
