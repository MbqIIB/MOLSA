Database Table: ProgramType

Table Description
ProgramType : Stores the configuration details about programs that a citizen can apply for. This entity dictates the behaviour of programs both on the Citizen Workspace and the internal worker application.

Example

An organization delivers Rent Allowance and Food Allowance programs to citizens. The organization configures ProgramType records and specifies the behaviour required for each.
