Database Table: ServicePlanApprovalCheck

Table Description
Service Plan Approval Check: Approval rules for a service plan.

Approval checks functionality is available for both a user and a service plan.

Example

The following service plan approval check can be created:

  • 10% of Return to Work service plans submitted for approval require manual approval by the supervisor of the user that submitted the service plan .
The remaining 90% of service plans will be automatically approved by the system.
