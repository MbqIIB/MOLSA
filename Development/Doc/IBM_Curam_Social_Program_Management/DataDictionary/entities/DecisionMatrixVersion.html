Database Table: DecisionMatrixVersion

Table Description
DADecision Matrix Version: A Decision Matrix defines the rules by which a determination configuration can reach a particular outcome. The decision matrix allows the user to define the rules used to a achieve outcomes.A Decision Matrix can have many versions over a period of time.

A Decision Matrix defines the rules by which a determination can reach outcomes. The decision matrix allows the user to define the set of rules used to achieve outcomes. A Decision Matrix can have many versions over a period of time.
