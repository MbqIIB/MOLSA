Database Table: MotivationType

Table Description
MotivationType : Clients are generally motivated to visit the Citizen Workspace in order to carry out some kind of activity related to applying for or receiving assistance from a Social Enterprise. A Motivation Type is a description of a process that can be executed by a client when they visit the Citizen Workspace. A Motivation Type is instantiated by a Motivation Entity. A Motivation Type is designed to help a client achieve a particular goal such as finding health care, submitting a life event or screening for a program.

In the main, the life cycle of a motivation conforms to the following basic pattern:

  • Collect data from the user
  • Execute rules based on the collected data
  • Display results based on the rules execution
  • Perform follow on actions

Example

Screening, Find Assistance (in HCR) and Life Events are examples of motivation types.
