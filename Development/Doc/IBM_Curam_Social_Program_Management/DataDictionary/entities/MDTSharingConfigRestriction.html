Database Table: MDTSharingConfigRestriction

Table Description
Multidisciplinary Team Sharing Configuration Restriction : Holds the sharing restriction details for a multidisciplinary team sharing configuration record. Defines the sharing restrictions specified for a sharing configuration record specified for sharing. That is, values ?preselected? or ?not preselected? have been selected as the sharing type on the related multidisciplinary team sharing configuration record. When a restriction record is created for a specified record of the sharing type the multidisciplinary member related to this restriction will not have access to information surrounding this restricted record.
