Database Table: CaseEvidenceLink

Table Description
Case Evidence Link : Link entity which connects specific evidence entities to the Case Evidence entity. Link entity which connects specific evidence entities to the Case Evidence entity.

Technical Notes

Please Note: This entity is no longer part of the default flow in the Cúram Application, as such it is neither written to nor read from by the Cúram Application. It continues to be shipped with the Cúram model as customer may have custom logic which uses this entity, however it should be noted that if you are implementing custom logic which references this entity then you must also ensure that it is written to as needed. The information that was originally stored in this entity is now stored in case evidence group link.
