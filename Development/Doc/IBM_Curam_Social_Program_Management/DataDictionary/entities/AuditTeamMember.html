Database Table: AuditTeamMember

Table Description
Audit Team Member : A relationship between an Auditor user and an AuditTeam record.

An audit team member represents an auditor on an audit team.

Example

The audit coordinator has added some auditors to the audit plan. The coordinator then chooses to create an audit team. The coordinator gives a name to the team and chooses the members from the list of current auditors on the audit plan.
