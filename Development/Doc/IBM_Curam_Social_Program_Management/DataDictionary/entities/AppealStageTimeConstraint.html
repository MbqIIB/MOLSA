Database Table: AppealStageTimeConstraint

Table Description
Time constraints set for a specific appeal stage. Time constraints can be added to various stages of an appeal. Each time constraint type has a time limit set (in number of days) in which it needs to be performed. For example, the time constraint type ?Decide and implement a Hearing? will define the number of days in which the ?Hearing? appeal stage must have a decision submitted and approved.
