Database Table: SORosterConfiguration

Table Description
Service Offering Roster Configuration: The attendance roster-related configuration settings for a service offering.

The attendance roster-related configuration settings for a service offering.

Example

The organization has set the 'Roster Grace Period' to 15 days for the Child Care service offering. If the roster submission date is 07/05/2008 then the 'Roster Due Date' for the roster of this service offering will be 15 days after this date, 07/20/2008.
