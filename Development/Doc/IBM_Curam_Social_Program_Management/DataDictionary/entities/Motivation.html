Database Table: Motivation

Table Description
Motivation : An instance of a Motivation Type Represents an instance of a MotivationType created on behalf of a user. A motivation instance is created by or on behalf of a client in order to achieve a particular goal such as finding health care, submitting a life event or screening for a program.

In the main, the life cycle of a motivation conforms to the following basic pattern:

  • Collect data from the user
  • Execute rules based on the collected data
  • Display results based on the rules execution
  • Perform follow on actions

The execution of the rule set leads to the creation of related MotivationResult.

Example

Screening, Find Assistance (in HCR) and Life Events are examples of motivations.
