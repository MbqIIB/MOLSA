Database Table: TaxonomyImportLog

Table Description
Taxonomy Import Log : Log that maintains history of changes to taxonomy term during import.

A log that is created when a taxonomy file is uploaded the first time or when an update happens. Details of the changes such as new terms included, terms updated and terms deleted are recorded.
