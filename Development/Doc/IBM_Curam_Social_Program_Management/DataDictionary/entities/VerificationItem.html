Database Table: VerificationItem

Table Description
Verification Item: A verification item is a document or activity that must be provided or undertaken in order to verify the accuracy of information obtained from the Curam client.

The means of verification can vary from provision of a document, e.g. birth certificate or bank statement, to a phone call.

Example

A Birth Certificate, Passport or Driver's License are all considered verification items. These verification items could be used to provide verification for a client's date of birth or gender.

A verification item does not necessarily represent an actual item or event. For instance, these may be the case where a user realizes that a client is not going to be eligible to receive a particular benefit. Rather than going through the formality of requesting verification, the user may just want to state that verification is unnecessary. Therefore, a verification item could be set up that effectively marks the verification as unnecessary. Security identifiers on the utilization of a verification item ensures that this facility is only available to appropriate users.
