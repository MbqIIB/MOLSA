Database Table: AbsenceReasonConfiguration

Table Description
Absence Reason Configuration: The configuration settings related to absence reason at both organization and service offering level.

The Configuration used to identify whether a absence reason is payable or non payable and deductible or non- deductible for a period. An absence reason configuration applies to all service offerings unless a configuration for an individual service offering is created to override it.

Example

The organization sets the payable indicator to false and deductible indicator to true for the 'Did Not Attend' at organization level. These indicator will be used when the provider mark the client absent for the above reason. The payment against the units of service for absent reason 'Did not attend' will not be made to the provider as the payable indicator is set to false but the units will be deducted from the total units authorized for the client.
