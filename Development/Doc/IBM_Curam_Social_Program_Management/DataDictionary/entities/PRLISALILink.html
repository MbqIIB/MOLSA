Database Table: PRLISALILink

Table Description
Provider Roster Line Item and Service Authorization Line Item Link: Associates the provider roster Line item to service authorization line item.

The association details between provider roster line item and service authorization line item.

Example

For example, a provider roster line item can span across many service authorization line items, for a client.
