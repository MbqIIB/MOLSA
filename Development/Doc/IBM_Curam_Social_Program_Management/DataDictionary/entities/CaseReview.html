Database Table: CaseReview

Table Description
Case Review: An examination of a case by a user.

A case review can be manually created by a user or automatically generated by the system.

Automatic case reviews are defined at a product level and can be scheduled to occur at defined frequencies e.g. every 6 months.

This table holds a type of case event. Events are displayed in an events calendar for viewing by the user.
