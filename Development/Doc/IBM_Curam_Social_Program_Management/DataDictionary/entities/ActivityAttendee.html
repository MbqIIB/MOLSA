Database Table: ActivityAttendee

Table Description
Activity Attendee: A participant invited to attend an activity.

For each attendee invited to an activity, the status of their reply e.g. Accepted, Provisional or Rejected is displayed to the user that scheduled the activity. When a user is invited to an activity, the invitation is displayed in their calendar. When a participant is invited to an activity, an email containing details of the activity is emailed to the participant.

Example

User Ronan Rooney creates an activity and invites two users to attend the meeting. Viewing the activity the following day, Ronan Rooney can see that one of the users has accepted the meeting invitation while the other has declined the invitation.
