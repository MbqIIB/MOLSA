Database Table: VerificationItemProvided

Table Description
Verification Item Provided: Indicates the details of the verification item provided or undertaken in order to confirm the accuracy of the evidence added to a case.

This entity records the details of the provision of verification item in order to satisfy a verification requirement.

Example

A case worker inserts personal details for case member James Smith on an integrated IV-A case. The personal details consist of a name and date of birth. There is a requirement to verify the date of birth for all IV-A cases with a level 1 verification item. There is a further requirement to verify the date of birth for a TANF case, which has been added to the IV-A case, with a minimum of two level 5 documents. James must supply at least a level 1 verification item in order to achieve the requirement for the IV-A case. If James supplies two level 5 documents, these will achieve the verification requirement verification requirementfor both the IV-A and TANF cases, and any subsequent usages that require up to a level 5 verification item. The provision of the documents is recorded using this entity.

The provision of verification items applies to all verification requirements for a verifiable data item.
