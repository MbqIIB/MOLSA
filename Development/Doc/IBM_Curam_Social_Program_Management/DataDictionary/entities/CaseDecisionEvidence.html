Database Table: CaseDecisionEvidence

Table Description
Case Decision Evidence: The set of evidence used to arrive at a case decision.

Example

The evidence associated with a sickness benefit claim could include:

  • a doctor's name;
  • the examination date; and
  • the date last worked.
