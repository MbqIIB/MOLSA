Database Table: CaseAudit

Table Description
Case Audit : A case audit is the individual case being audited as part of an audit plan.

This entity captures the current state of a case audit and its associated Case and audit plan.

Example

When a case is initially added to an audit plan, a CaseAudit record is created for it on that plan and the status is set to Open. The case audit can progress through many statuses including 'Assigned', 'Findings Complete', 'Findings Complete & Awaiting Feedback', 'Feedback Received', 'Feedback Complete' and finally 'Complete'.
