Database Table: EvidenceMetadata

Table Description
Evidence Metadata: Configuration data about an evidence type.

The evidence metadata entity stores configuration data about an evidence type. The configuration data includes the names of the Curam application pages to be used for displaying and editing evidence. Therefore this configuration data is used during standard temporal evidence functions, e.g. view and modify.

Example

The configuration data stored about an evidence type includes the names of the Curam application pages to be used for displaying and editing evidence, e.g., the Curam application page to use for viewing evidence of this type.
