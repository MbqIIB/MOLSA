Database Table: BizObjAssociation

Table Description
Business Object Association : An association of a Task with a row on a database table.

This table allows a Task to be related to any other row of database data, e.g.:

  • a Person; or
  • a Case;

This facility enables Task functionality to be extended and permits custom application functionality to tie any required information to a Task.

Example

A Task is created which relates to one of Person John Smith's Cases.

Two rows are created on this entity:

  • one to associate the Task with Person John Smith; and
  • one to associate the Task with John's Case.

These rows allow the Task to be listed both on John's Person Home Page and on the Case Home Page.
