Database Table: AssessmentDeliveryRole

Table Description
Assessment Delivery Role : The role a client plays on an assessment.

When running an assessment a user must select the roles a client plays in the assessment. The configurations specified on the assessment roles entity govern which roles must be and can be selected by a case worker.

Example

James Smith, Linda Smith and their child Robert must be assessed to identify the family's strengths and needs in an effort to help build an effective Reunification outcome plan. The Family Strengths and Needs Assessment (FSN) is run from the family's outcome plan by a case worker. Linda is selected as Roberts primary caregiver, James is his secondary caregiver and Robert is selected as the focus child. This entity stores the roles of primary caregiver, secondary caregiver and focus child.
