Database Table: TemporalEvidenceApprovalCheck

Table Description
Temporal Evidence Approval Check: Maintains configuration details for approval check functionality associated with evidence maintained using the temporal evidence solution.

Temporal Evidence Approval Checks specifies the percentage of evidence that must be manually approved, for evidence which is maintained using the temporal evidence solution. Temporal Evidence Approval Checks can be specified at the User, Organization Unit, Position and Evidence Type level.

Example

A job of junior case worker exists within an organization. An organization dealing with income support cases holds one such position of this job type. The organization decides that due to the inexperienced level of any user holding this position, a large number of evidence changes worked on by such users should be routed for manual approval and therefore the organization configure evidence approval checks at 75% for this position. During evidence maintenance, for any user holding this position, 75% of the evidence changes they submit will be routed to the case supervisor for manual approval. The remaining 25% of evidence changes will be automatically approved.
