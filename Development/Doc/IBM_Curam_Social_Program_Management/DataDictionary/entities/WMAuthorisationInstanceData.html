Database Table: WMAuthorisationInstanceData

Table Description
WMAuthorisationInstanceData : The additional data required for the deferred authorization process. This entity captures details relating to deferred authorization processing. The authorization may be for a specific program application or the authorization of the entire application case.

Example

An intake worker chooses to authorize a program application. They are presented with the ability to add their comments and, based on the configuration, select the on-going integrated case and product delivery clients. On completion of this task control is returned to the user. In the background the system attempts to authorize the program application. If successful the status of the intake program application is set to Approved, if unsuccessful the intake worker is notified.
