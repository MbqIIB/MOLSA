Database Table: AdministrationRole

Table Description
Administration Role: A user assigned to manage the interactions between the organization and a participant role.

By default the user that registers a participant is set as the administrator of the participant.

Example

User Ronan Rooney is assigned as the administrator for the participant John Smith. Ronan Rooney is therefore responsible for managing all interaction between the organization and John Smith e.g. handling queries, correspondence, etc.
