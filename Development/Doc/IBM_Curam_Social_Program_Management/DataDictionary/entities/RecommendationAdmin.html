Database Table: RecommendationAdmin

Table Description
Recommendation Administration : Stores service, action and referral recommendations for a factor classification or a factor classification subdivision.

Recommended activities are displayed to a case worker on the activity workspace and are based on a client scoring a particular classification/score when assessed for a factor.

Example

An outcome plan is created for James Smith, Linda Smith and their child Robert to identify and address their issues. The Family Strengths and Needs Assessment (FSN) Assessment is run from this outcome plan by a case worker. James Smith is classified as having a Need in Substance Abuse and as such a counseling service is recommended by the system. Linda Smith is classified as a Need in Parenting Skills and as such a Parenting Skills class is recommended by the system. This entity allows for these activities to be recommended.
