Database Table: ActivityAttendeeExt

Table Description
Activity Attendee Extension : Extension of the existing Activity entity to provide additional fields for meeting processing.

Please Note: This entity is no longer part of the default flow in the Cúram Application (and has not been since version 6), as such it is neither written to nor read from by the Cúram Application. It continues to be shipped with the Cúram model as customer may have custom logic which uses this entity, however it should be noted that if you are implementing custom logic which references this entity then you must also ensure that it is written to as needed. The information that was originally stored in this entity is now stored in Activity itself.
