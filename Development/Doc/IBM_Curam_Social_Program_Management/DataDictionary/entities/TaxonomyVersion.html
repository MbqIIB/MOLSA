Database Table: TaxonomyVersion

Table Description
Taxonomy Version : Taxonomy version details points to taxonomy file attachment and related information.

Information for a taxonomy file import. Also information such as whether the taxonomy has been uploaded the first time or whether the existing list is updated with a new file, date and time of upload etc., are logged during the import.
