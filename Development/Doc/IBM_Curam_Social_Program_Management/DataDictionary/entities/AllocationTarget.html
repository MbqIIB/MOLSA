Database Table: AllocationTarget

Table Description
Allocation Target: A container for allocation target items.

Task allocation is the assignment of a task to either a work queue, or any of the organization structure elements:

  • organization unit;
  • position;
  • job; and
  • user.

Tasks are targeted (assigned) to all target items added to an allocation target.
