Database Table: ServiceInvoice

Table Description
Service Invoice: A service invoice contains summary information entered by a user about the service invoice line items that it contains, as well as information that need only be captured once for all service invoice line items.

A service invoice contains summary information entered by a user about the service invoice line items that it contains, as well as information that need only be captured once for all service invoice line items.

Example

For example, if one provider is invoicing the agency for all services contained on the service invoice line items, then information about the provider can be maintained at the service invoice level.
