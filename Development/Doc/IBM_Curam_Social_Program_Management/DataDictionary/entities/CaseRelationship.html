Database Table: CaseRelationship

Table Description
Case Relationship: An association between one case and another.

Each row on this table links two related cases. Related cases can be automatically or manually created.

Example

A case relationship can be created where an overpayment case occurs. The overpayment case is therefore related to the case on which the overpayment occurred.
