Database Table: EvidenceTypeDef

Table Description
Evidence Type Definition: This entity defines a dynamic evidence type.

This entity does not contain the actual dynamic evidence metadata information but its child entity EvidenceTypeVersionDef does. This entity is the link point which allows products and integrated cases to use the dynamic evidence type.
