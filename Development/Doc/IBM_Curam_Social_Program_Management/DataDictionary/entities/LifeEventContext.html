Database Table: LifeEventContext

Table Description
Life Event Context : A type of life event in a particular context(channel in the application), for example a Change of Address life event for a citizen using the citizen self service. Life events can be configured differently depending on the channel in which they are used, for example, the question script displayed to a citizen in the citizen self service may be different to that displayed to a case worker in the internal application. In this case, this entity allows for different scripts, one for the citizen and one for a case worker.

Example

James Smith logs into the citizen account and selects to submit a change of address life event. James selects to submits his new address. The question script that is displayed to the citizen is stored on this entity.
