Database Table: PRLITransaction

Table Description
Provider Roster Line Item Transaction: Transaction details for a provider roster line item.

A transaction on a roster line item that occurs during the processing of the roster line item that affects the final amount that will be approved for payment.

Example

For example, if a provider roster line item is submitted for processing and validated successfully, a transaction with transaction type of 'Paid' and with the final payment amount against the provider roster line item is created. If a provider roster line item is denied or canceled, a transaction with corresponding transaction type is created.
