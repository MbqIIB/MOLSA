Database Table: RemoteSystem

Table Description
A system which can send/receive data via a web service. Applications and life events can be sent to systems other than Curam. Messages, payments and contact information which have come from systems other than Curam can be displayed in the citizen account. This entity is used in conjunction with the Target System entity to define these external/remote systems.
