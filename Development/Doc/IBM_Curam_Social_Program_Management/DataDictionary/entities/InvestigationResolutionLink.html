Database Table: InvestigationResolutionLink

Table Description
Investigation Resolution Link: A relationship between a resolution and a type of investigation.

This table allows for the associations between a resolution and a type of investigation in Investigation Administration.

Example

During case processing a user receives information that a client may be working while at the same time claiming benefit. An investigation of type Benefit Fraud is be created by the user to record details of the suspicion and any actions taken to determine if the client is at fault. A supervisor reviews the investigation and selects an appropriate resolution on the investigation based on all the information recorded.

To facilitate this, the following needs to be created in the Administration component:

  • An administrator creates an investigation of type Benefit Fraud.
  • A resolution of 'Founded' is also created.
  • This resolution is associated with the benefit fraud investigation.
This table stores the association between the resolution and the benefit fraud investigation.
