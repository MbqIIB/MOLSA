Output Formats

Introduction

DocMaker can convert DocBook XML content into a number of different formats. At present, the generation of output in the portable document format (PDF) and the HTML Help format are officially supported and actively maintained. While DocMaker can also generate plain HTML output, this feature is not yet officially supported.

Your content in DocBook XML format is used as the source for all output formats: you do not need to maintain different content to support each output format. Generating output in different formats is as simple as running the appropriate DocMaker target: generate-pdf for PDF and generate-help for HTML Help. However, as these output formats fulfill different purposes and are used in different ways, DocMaker includes some features to help you get the most out of your content: you can make your help system context-sensitive, and you can include or exclude content depending on the output format selected.
