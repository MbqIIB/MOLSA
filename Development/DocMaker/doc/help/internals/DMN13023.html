Prerequisites

Before converting documents, you must have both DocMaker and FrameMaker installed on your computer. FrameMaker must be configured to use the “Structured FrameMaker” interface. This can be selected at installation time, or, if already installed, by selecting File→Preferences→General... and changing the Product Interface to “Structured FrameMaker” before pressing the Set button. FrameMaker will need to be restarted for this change to take effect.
