The Conversion Process

Cúram documents are organized as books with separate files for the front matter, table of contents, and the main text. The text may be organized with one or more chapters or appendices per file. Books organized in a different manner cannot be converted.

Only the main text is converted, the front matter is not processed and is discarded. Most of the discarded text is just boilerplate or generated information that DocMaker will create afresh. Information such as non-standard legal notices and the printing history must be recreated by hand after the conversion process.

The conversion process is performed in several phases:

  1. Each file in the book must first be checked and prepared for conversion. If a document is in good order, this process is relatively quick and painless. However, documents that do not use the Cúram template correctly may take some time to prepare.

  2. The conversion facilities provided by FrameMaker are used to add structure to the documents. A FrameMaker document called a conversion table is used to identify known sequences of paragraph formats and map them to DocBook-like[3] elements. If the paragraph formats were used in unconventional ways, there will be problems identifying the documents' structure.

  3. A simple FrameMaker structure application is used to control the creation of XML files from the structured documents. The structure application defines the export rules for any images included in the document.

  4. A temporary book.xml file is created to group and order the XML files created by FrameMaker. As FrameMaker cannot provide a meaningful conversion of its book file, this file is created by DocMaker and then adjusted by hand as needed.

  5. DocMaker takes the XML files produced by FrameMaker, and the book file that groups them, and converts it to DocBook XML format. This conversion process performs some integrity checks, resolves cross-references, and restructures the FrameMaker output into a valid DocBook document.

  6. Once the DocBook is produced, you must perform some manual steps to finish the conversion process. This includes the renaming of image files and the completion of the main book.xml file.
