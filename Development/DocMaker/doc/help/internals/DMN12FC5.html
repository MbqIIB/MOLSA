Ant Script Profiling

The same type of CSV profiles that are generated by the Transform task to profile XSLT stylesheets can also be generated to profile the execution of entire Ant scripts. An Ant build listener class is included with DocMaker that will generate a profile in a CSV file if the listener is defined when invoking Ant. Two JAR files, profile-ant.jar and jde-commons-ext.jar, are required when profiling Ant scripts. They should be on the system classpath supplied to Ant. This can be achieved in two ways: copy these files to the lib directory of the Ant installation, or include then in the CLASSPATH environment variable. The latter method is less invasive. To enable the profiling, use Ant's -listener command-line option to include curam.docmaker.tools.profile.ProfilingAntListener as a build listener. After the Ant build has completed, the profile will be written to a file named ant-profile.csv.
