DocMaker Tools

Introduction

DocMaker has a number of tools that may be useful in other projects. In particular, it includes an Ant task for performing XSLT transformations and facilities for profiling the performance of stylesheets and Ant build scripts. These tools can be borrowed from DocMaker and reused in other projects.
