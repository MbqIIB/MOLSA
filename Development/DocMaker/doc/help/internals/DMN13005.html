Converting FrameMaker Documents

Introduction

Originally, Cúram documentation was written using Adobe FrameMaker. DocMaker provides a number of facilities to aid the conversion of documents in FrameMaker format to the DocBook XML format. While much of the conversion process is automatic, manual intervention is necessary and the quality of the output depends on a number of factors that will be described in detail. Be comforted by the fact that you only have to do this once!
