Cúram DocMaker Internals

Abstract

This manual describes the installation and operation of DocMaker, Cúram's documentation production environment. The DocBook XML format used by DocMaker is presented along with detailed information on the generation of documentation in PDF and HTML Help format. Information on the integration of DocMaker with other build systems is also provided.
