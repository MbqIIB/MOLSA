Database SQL Operation: InvestigationDelivery.readHomePageIDInvestigationTypeCaseRef

Operation Details
Description: Reads the home page identifier, investigation type and case reference for a Investigation Delivery case.
Sterotype: ns

SQL 
SELECT
  InvestigationConfig.homePageIdentifier,
  InvestigationDelivery.investigationType,
  CaseHeader.caseReference
INTO
  :homePageIdentifier,
  :investigationType,
  :caseReference 
FROM
  InvestigationConfig,
  InvestigationDelivery,
  CaseHeader 
WHERE InvestigationDelivery.caseID = :caseID 
  AND InvestigationConfig.investigationConfigID = InvestigationDelivery.investigationConfigID 
  AND CaseHeader.caseID = InvestigationDelivery.caseID
