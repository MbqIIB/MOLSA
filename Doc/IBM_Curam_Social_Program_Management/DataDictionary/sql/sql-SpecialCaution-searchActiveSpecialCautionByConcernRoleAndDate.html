Database SQL Operation: SpecialCaution.searchActiveSpecialCautionByConcernRoleAndDate

Operation Details
Description: Method&nbsp;to&nbsp;search&nbsp;active&nbsp;Special Caution Details&nbsp;list by concern role ID and date.
Sterotype: nsmulti

SQL 
SELECT DISTINCT
  SpecialCaution.specialCautionTypeCode,
  SpecialCaution.startDate
INTO
  :specialCautionTypeCode,
  :startDate
FROM
  SpecialCaution 
WHERE SpecialCaution.concernRoleID = :concernRoleID 
  AND SpecialCaution.recordStatus = :recordStatus 
  AND SpecialCaution.startDate <= :currentDate
  AND (SpecialCaution.endDate IS NULL OR SpecialCaution.endDate > :currentDate)
