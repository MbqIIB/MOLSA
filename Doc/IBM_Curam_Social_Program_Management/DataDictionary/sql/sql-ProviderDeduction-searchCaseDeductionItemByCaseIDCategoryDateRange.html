Database SQL Operation: ProviderDeduction.searchCaseDeductionItemByCaseIDCategoryDateRange

Operation Details
Description: Retrieves the Case Deduction Items using the case ID, category,deduction name, start date and end date as the parameters.
Sterotype: nsmulti

SQL 
SELECT
  CaseDeductionItem.caseDeductionItemID,
  CaseDeductionItem.concernRoleID,
  CaseDeductionItem.createdDate,
  CaseDeductionItem.startDate,
  CaseDeductionItem.endDate,
  CaseDeductionItem.statusCode,
  CaseDeductionItem.comments,
  CaseDeductionItem.relatedCaseID,
  CaseDeductionItem.caseID,
  CaseDeductionItem.rate,
  CaseDeductionItem.amount,
  CaseDeductionItem.relatedObjectiveID,
  CaseDeductionItem.rulesObjectiveID,
  CaseDeductionItem.deductionType,
  CaseDeductionItem.deductionName,
  CaseDeductionItem.thirdPartyConcernRoleID,
  CaseDeductionItem.customerAccName,
  CaseDeductionItem.customerAccNumber,
  CaseDeductionItem.priority,
  CaseDeductionItem.category,
  CaseDeductionItem.businessStatus,
  CaseDeductionItem.actionType,
  CaseDeductionItem.caseNomineeID,
  CaseDeductionItem.deductionID,
  CaseDeductionItem.versionNo
INTO
  :caseDeductionItemID,
  :concernRoleID,
  :createdDate,
  :startDate,
  :endDate,
  :statusCode,
  :comments,
  :relatedCaseID,
  :caseID,
  :rate,
  :amount,
  :relatedObjectiveID,
  :rulesObjectiveID,
  :deductionType,
  :deductionName,
  :thirdPartyConcernRoleID,
  :customerAccName,
  :customerAccNumber,
  :priority,
  :category,
  :businessStatus,
  :actionType,
  :caseNomineeID,
  :deductionID,
  :versionNo 
FROM
  CaseDeductionItem 
WHERE CaseDeductionItem.caseID = :caseID 
  AND CaseDeductionItem.category = :category 
  AND CaseDeductionItem.deductionName = :deductionName 
  AND CaseDeductionItem.startDate = :startDate 
  AND 
  (
    CaseDeductionItem.endDate = :endDate 
    OR CaseDeductionItem.endDate IS NULL 
  )
